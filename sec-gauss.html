    Section 1.3 Gauss--Jordan elimination

    By this point I hope that you are convinced that solutions to linear systems in reduced echelon form are easy to find. The question now is whether and how we can put a linear system in reduced echelon form. We can always do so using elementary row operations. The process is known as Gauss-Jordan elimination.
    ðŸ”—
    This process consists of several steps. I will describe them now in full generality, but will use a concrete matrix so that we can see how the matrix is changed along the way. Since the purpose of this algorithm is to solve systems of linear equations, I will also display alongside the matrix the corresponding system of linear equations that we get at each step. In practice, we would not write the linear system every step of the way. We would work solely with the matrix, and we would translate the information in the last matrix back in terms of the variables in the initial system.
    ðŸ”—
    The linear system/matrix pair that we will use is
    \begin{equation*} \left\{ \begin{alignedat}{9} 3y \aa - \aa 2z \aa + \aa w \amp = \aa 0\\ 2x \aa + \aa 3y \aa + \aa z \amp = \aa 1\\ \aa \aa 3x \aa + \aa 2w \amp = \aa 3 \end{alignedat}\right.,\qquad\qquad\qquad \left( \begin{array}{cccc|c} 0 \amp 3 \amp -2 \amp 1 \amp 0\\ 2 \amp 3 \amp 1 \amp 0 \amp 0\\ 3 \amp 0 \amp 0 \amp 2 \amp 3\\ \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Step 1. Start by identifying the leftmost column that does not consist entirely of zeroes. If the top entry of that column is a \(0\text{,}\) swap the top row with the first row which has a non-zero entry in that column.
    ðŸ”—
    In the augmented matrix above, that is column \(1\text{.}\) Note that the top entry in column \(1\) is \(0\) which means that we will need to swap the top row with some another row in order to make the first entry in column \(1\) be non-zero. Letâ€™s go ahead and swap the first two rows. We end up with
    \begin{equation*} \left\{ \begin{alignedat}{7} 2x \aa + \aa 3y \aa + \aa z \amp = \aa 1\\ 3y \aa - \aa 2z \aa + \aa w \amp = \aa 0\\ \aa \aa 3x \aa + \aa 2w \amp = \aa 3\\ \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 2 \amp 3 \amp 1 \amp 0 \amp 1\\ 0 \amp 3 \amp -2 \amp 1\amp 0\\ 3 \amp 0 \amp 0 \amp 2 \amp 3\\ \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Step 2. The top entry in the column identified in step \(1\) is non-zero. If that entry is a number \(a\neq 1\text{,}\) divide the whole top row by \(a\text{.}\)
    ðŸ”—
    The first entry in the first column is a \(2\text{,}\) so we will need to divide the top row by \(2\text{.}\) After performing this operation, we obtain
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp = \aa \frac{1}{2}\\ 3y \aa - \aa 2z \aa + \aa w \amp = \aa 3\\ \aa \aa 3x \aa + \aa 2w \amp = \aa 3 \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\ 0 \amp 3 \amp -2 \amp 1 \amp 0\\ 3 \amp 0 \amp 0 \amp 2 \amp 3 \end{array} \right)\text{.} \end{equation*}
    The first entry in the first column is a \(2\text{,}\) so we will need to divide the top row by \(2\text{.}\) After performing this operation, we obtain
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp \aa \frac{1}{2}\\ 3y \aa - \aa 2z \aa + \aa w \amp = \aa 0\\ \aa \aa 3x \aa + \aa 2w \amp = \aa 3\\ \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\ 0 \amp 3 \amp -2 \amp 1 \amp 0\\ 3 \amp 0 \amp 0 \amp 2 \amp 3 \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Step 3. The top row now has a leading \(1\text{.}\) Add multiples of the top row to the rows below to ensure that the only non-zero entry in the column of steps \(1\) and \(2\) is the leading \(1\) in the top row.
    ðŸ”—
    Currently, our matrixâ€™s first column has a \(1\) in the first entry, a \(0\) in the second, and a \(3\) in the last entry. We need to get rid of that \(3\text{.}\) In order to do that, we take the last row and remove \(3\) times the first row. We end up with
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp = \aa \frac{1}{2}\\ 3y \aa - \aa 2z \aa + \aa w \amp = \aa 0\\ -\frac{9}{2}y \aa - \aa \frac{3}{2}z \aa + \aa 2w \amp = \aa \frac{3}{2} \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\ 0 \amp 3 \amp -2 \amp 1 \amp 0\\ 0 \amp -\frac{9}{2} \amp -\frac{3}{2} \amp 2 \amp \frac{3}{2} \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Step 4. Now consider the submatrix obtained from the original matrix after removing the top row. Perform steps \(1\) to \(4\) on that submatrix.
    ðŸ”—
    This is usually the longest step. In our example, the submatrix in question is \(\left(\begin{array}{ccc|c}0 \amp 3 \amp -2 \amp 1 \amp 0\\0 \amp -\frac{9}{2} \amp -\frac{3}{2} \amp 2 \amp \frac{3}{2}\end{array}\right)\text{.}\) The leftmost column with non-zero entries is column \(2\text{,}\) and the top row of that column is already non-zero. This means that we can go immediately to step \(2\) and divide the top row of the submatrix (second row of original matrix) by \(3\text{.}\) We get
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp = \aa \frac{1}{2}\\ y \aa - \aa \frac{2}{3}z \aa + \aa \frac{1}{3}w \amp = \aa 0\\ -\frac{9}{2}y \aa - \aa \frac{3}{2}z \aa + \aa 2w \amp = \aa \frac{3}{2} \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp \frac{1}{3} \amp 0\\[6pt] 0 \amp -\frac{9}{2} \amp -\frac{3}{2} \amp 2 \amp \frac{3}{2} \end{array} \right)\text{.} \end{equation*}
    Now we will need to add $9/2$ times the second row of our matrix to the last row. After doing that, we obtain
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp = \aa \frac{1}{2}\\ y \aa - \aa \frac{2}{3}z \aa + \aa \frac{1}{3}w \amp = \aa 0\\ \aa - \aa \frac{9}{2}z \aa + \aa \frac{7}{2}w \amp = \aa \frac{3}{2} \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp \frac{1}{3} \amp 0\\[6pt] 0 \amp 0 \amp -\frac{9}{2} \amp \frac{7}{2} \amp \frac{3}{2} \end{array} \right)\text{.} \end{equation*}
    Finally, we need to multiply the last row of the matrix by \(-2/9\) to get
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{1}{2}z \amp = \aa \frac{1}{2}\\ y \aa - \aa \frac{2}{3}z \aa + \aa \frac{1}{3}w \amp = \aa 0\\ \aa \aa z \aa - \aa \frac{7}{9}w \amp = \aa -\frac{1}{3} \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp \frac{1}{2} \amp 0 \amp \frac{1}{2}\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp \frac{1}{3} \amp 0\\[6pt] 0 \amp 0 \amp 1 \amp -\frac{7}{9} \amp -\frac{1}{3} \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Step 5. Starting in the last row and working upward, add a suitable multiple of each of the lower rows to the ones above to enure that each pivot column only has a \(1\) in the pivot position and \(0\)s everywhere else.
    ðŸ”—
    We will first add \(2/3\) times the last row to the second row, and \(-1/2\) times the last row to the first one. The new system and matrix that we get are
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa \frac{3}{2}y \aa + \aa \frac{7}{18}w \amp = \aa \frac{2}{3}\\ \aa \aa y \aa + \aa \frac{13}{18}w \amp = \aa \frac{1}{6}\\ \aa \aa z \aa - \aa \frac{7}{9}w \amp = \aa -\frac{1}{3} \end{alignedat} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp \frac{3}{2} \amp 0 \amp \frac{7}{18} \amp \frac{2}{3}\\[6pt] 0 \amp 1 \amp 0 \amp \frac{13}{18} \amp \frac{1}{6}\\[6pt] 0 \amp 0 \amp 1 \amp -\frac{7}{9} \amp -\frac{1}{3} \end{array} \right)\text{.} \end{equation*}
    Now we only need to add \(-3/2\) times the second row to the first row to get the linear system in reduced echelon form that we were seeking:
    \begin{equation*} \left\{ \begin{aligned} x-\frac{25}{36}w \amp =\frac{5}{12}\\ y+\frac{13}{18}w \amp =\frac{1}{6}\\ z-\frac{7}{9}w \amp =-\frac{1}{3} \end{aligned} \right.\qquad\qquad\qquad \left( \begin{array}{cccc|c} 1 \amp 0 \amp 0 \amp -\frac{25}{36} \amp \frac{5}{12}\\[6pt] 0 \amp 1 \amp 0 \amp \frac{13}{18} \amp \frac{1}{6}\\[6pt] 0 \amp 0 \amp 1 \amp -\frac{7}{9} \amp -\frac{1}{3} \end{array} \right)\text{.} \end{equation*}
    ðŸ”—
    Now that we have reduced the system to one the reduced echelon form, we can easily find all the solutions. First note that there are three leading variables, \(x,y\) and \(z\text{,}\) and one free variable, \(w\text{.}\) This means that all the solutions to this system of linear equations are given parametrically by
    \begin{equation*} x=\frac{5}{12}+\frac{25}{36}t,\quad y=\frac{1}{6}-\frac{13}{18}t,\quad z=-\frac{1}{3}+\frac{7}{9}t,\quad w=t \end{equation*}
    where \(t\in\R\text{.}\)
    ðŸ”—
    The process of Gauss-Jordan elimination can be used to show the following theorem.
    ðŸ”—

    Theorem 1.3.1. Gauss-Jordan elimination.

    A linear system with \(m\) variables and \(n\) equations is equivalent to a linear system with \(m\) variables and \(n\) equations in reduced echelon form. Moreover, a homogeneous linear system with $m$ variables and $n$ equations is equivalent to a homogeneous linear system with $m$ variables and $n$ equations.
    ðŸ”—
    ðŸ”—

    Example 1.3.2.

    Letâ€™s use Gauss-Jordan elimination to solve
    \begin{equation*} \left\{ \begin{alignedat}{7} x \aa + \aa y \aa - \aa z \amp = \aa 0\\ \aa \aa 2x \aa - \aa y \amp = \aa 1\\ \aa \aa 2x \aa + \aa 2z \amp = \aa 3 \end{alignedat} \right.\text{.} \end{equation*}
    The corresponding augmented matrix is
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp -1 \amp 0\\ 2 \amp -1 \amp 0 \amp 1\\ 2 \amp 0 \amp 2 \amp 3 \end{array} \right)\text{.} \end{equation*}
    First, we subtract \(2\) times the first row from the second and third rows to get:
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp -1 \amp 0\\ 0 \amp -3 \amp 2 \amp 1\\ 0 \amp -2 \amp 4 \amp 3 \end{array} \right)\text{.} \end{equation*}
    Next, we multiply the second row by \(-1/3\text{:}\)
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp -1 \amp 0\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp -\frac{1}{3}\\[6pt] 0 \amp -2 \amp 4 \amp 3 \end{array} \right)\text{.} \end{equation*}
    Adding ,\(2\) times the second row to the last one, we obtain
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp -1 \amp 0\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp -\frac{1}{3}\\[6pt] 0 \amp 0 \amp \frac{8}{3} \amp \frac{7}{3} \end{array} \right)\text{.} \end{equation*}
    Now we multiply the last row by \(3/8\text{:}\)
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp -1 \amp 0\\[6pt] 0 \amp 1 \amp -\frac{2}{3} \amp -\frac{1}{3}\\[6pt] 0 \amp 0 \amp 1 \amp \frac{7}{8} \end{array} \right)\text{.} \end{equation*}
    We are almost there. We now need to add \(2/3\)$ times the last row to the second one and add the last row to the first row.
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 1 \amp 0 \amp \frac{7}{8}\\[6pt] 0 \amp 1 \amp 0 \amp \frac{1}{4}\\[6pt] 0 \amp 0 \amp 1 \amp \frac{7}{8} \end{array} \right)\text{.} \end{equation*}
    Finally, we only need to subtract the second row from the first, and we get the following augmented matrix in reduced echelon form:
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 0 \amp 0 \amp \frac{5}{8}\\[6pt] 0 \amp 1 \amp 0 \amp \frac{1}{4}\\[6pt] 0 \amp 0 \amp 1 \amp \frac{7}{8} \end{array} \right)\text{.} \end{equation*}
    from where we deduce that the only solution to our linear system is \(x=\frac{5}{8}, y=\frac{1}{4}, z=\frac{7}{8}\text{.}\)

    Theorem 1.3.3.

    A homogeneous linear system with more variables than equations has infinitely many solutions.
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(m\) be the number of variables and \(n\) the number of equations. We have \(m>n\text{.}\) We know from TheoremÂ 1.3.1 that this system is equivalent to a homogeneous linear system with \(m\) equations and \(n\) variables in reduced echelon form. So all we need to do is to show that a homogeneous linear system in reduced echelon form that has more variables than equations admits infinitely many solutions. It will suffice to show that such a system has at least one free variable.
    ðŸ”—
    Since such a system is homogeneous, it must be consistent, i.e. it must have at least one solution, namely the trivial solution where all the variables are assigned the value \(0\text{.}\) From LemmaÂ 1.2.12 we know that this system has at most $n$ leading variables, and so the remaining variables are free. By assumption, the number of variables \(m\) is strictly larger than \(n\) so our linear system has at least one free variable, and so will admit infinitely many solutions.
