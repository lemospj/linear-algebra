    Section 2.4 Linear subspaces

    The notion of a linear subspace of a vector space is a central notion in linear algebra.
    ðŸ”—

    Definition 2.4.1.

    A subset \(V\) of \(\R^n\) is called a linear subspace of \(\R^n\) if the following conditions hold:
    1. If \(\bv,\bu\in V\text{,}\) then \(\bv+\bu\in V\) (i.e., \(V\) is closed under addition).
      ðŸ”—
      ðŸ”—
    2. If \(\lambda\in \R\) and \(\bv\in V\text{,}\) then \(\lambda\bv\in \R\) (i.e., \(V\) is closed under scalar multiplication).
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 2.4.2.

    Consider the set \(V=\{(x,y)\in\R^2|y=3x\}\subseteq \R^2\text{.}\) This set is a line in the plane containing the origin (the line \(y=3x\)), and it is an example of a linear subspace of \(\R^2\text{.}\) Indeed, if \(\bv=(x,3x)\in\R^2\text{,}\) then \(\lambda\bv=(\lambda x, 3\lambda x)\text{,}\) and it is clear that the \(y\)-coordinate of \(\lambda v\) is three times its \(x\)-coordinate, which means that \(\lambda\bv\in V\) for every \(\lambda\in\R\text{.}\) Similarly, if \(\bv=(x,y)\) and \(\bu=(s,t)\) are both in \(V\text{,}\) then we must have \(y=3x\) and \(t=3s\text{.}\) This leads to
    \begin{equation*} \bv + \bu = (x,3x)+(s,3s)=(x+s,3(x+s))\in V\text{.} \end{equation*}
    The two axioms of a linear subspace are satisfied, which means that \(V\) is a linear subspace of \(\R^2\text{.}\)

    ðŸ”—

    Example 2.4.3.

    The line \(y=3x+1\) is not a vector space. For example, \((0,1)\) is a vector of the form \((x, 3x+1)\text{,}\) but \(2(0,1)=(0,2)\) is not.

    One interesting example of linear subspaces comes from solution sets of homogeneous linear systems of equations. Indeed, every solution set of a homogeneous linear system of equations is a linear subspace. (See exercises.)
    ðŸ”—
    If \(V\) is a linear subspace of \(\R^n\) and \(S\) is a subset of vector in \(V\text{,}\) we say that \(S\) generates \(V\) if every vector in \(\bv\) can be written as a linear combination of vectors in \(S\text{.}\)

    Definition 2.4.7.

    Let \(S\) be a set of vectors in \(\R^n\text{.}\) The span of \(S\text{,}\) denoted \(\vspan(S)\text{,}\) is the set of vectors in \(\R^n\) that can be written as a linear combination of elements in \(S\text{.}\) We make the convention that the span of the empty set is the linear subspace of \(\R^n\) consisting of the zero vector.
    ðŸ”—
    ðŸ”—

    Proposition 2.4.8.

    Let \(S\) be a set of vectors in \(\R^n\text{.}\) The set \(\vspan(S)\) is a linear subspace of \(\R^n\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(\bv\) be an element in \(\vspan(S)\text{.}\) First, we will see that \(\lambda\bv\in\vspan(S)\) for every scalar \(\lambda\in \R\text{.}\)
    ðŸ”—
    By definition, every element \(v\in \vspan(S)\) can be written as a linear combination of vectors in \(S\text{:}\)
    \begin{equation*} v=\sum_{\bu\in S}c_{\bu}\bu\text{,} \end{equation*}
    where only finitely many of the coefficients \(c_u\) are non-zero. If \(\lambda\in\R\text{,}\) then
    \begin{equation*} \lambda\bv=\sum_{\bu\in S}\lambda c_{\bu}\bu\text{,} \end{equation*}
    which means that \(\lambda\bv\) can also be written as a linear combination of vectors in \(S\text{.}\) Therefore, \(\lambda\bv\) is in the span of \(S\text{.}\)
    ðŸ”—
    We now need to show that if \(\bv\) and \(\bw\) are two vectors in the span of \(S\text{,}\) then \(\bv+\bw\) will also be there.
    ðŸ”—
    Using the definition of span again, we know that \(\bv\) and \(\bw\) can be written as linear combinations of vectors in \(S\text{:}\)
    \begin{equation*} \bv = \sum_{\bu\in S}a_{\bu}\bu \qquad\text{and}\qquad \bw=\sum_{\bu\in S}b_{\bu}\bu\text{.} \end{equation*}
    Then
    \begin{equation*} \bv + \bw = \sum_{\bu\in S}(a_{\bu}+b_{\bu})\bu\text{,} \end{equation*}
    showing that \(\bv+\bw\) can also be written as a linear combination of vectors in \(S\text{.}\) This means that \(\bv+\bw\in\vspan(S)\) and concludes the proof that \(\vspan(S)\) is a linear subspace of \(\R^n\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.9.

    Consider the vector \(\bv=\colvec{1 \\ 2}\) in \(\R^2\text{.}\) All linear combinations of the set formed by this single vector are of the form \(\lambda\bv\) for some scalar \(\lambda\in \R\text{.}\) Therefore, \(\vspan(\bv)\) is simply the line in \(\R^2\) that passes through the origin and has the same direction as the vector \(\bv\text{.}\)
    ðŸ”—
    Consider now the set \(S\) formed by the vectors \(\bv=\colvec{1 \\ 2}\) and \(\bw=\colvec{-2 \\ -4}\text{.}\) Since \(\bw=-2\bv\) the span \(\vspan(\bv,\bw)\) is the same as \(\vspan(\bv)\text{.}\) Indeed, if a vector \(\bu\in\R^2\) is a linear combination of \(\bv\) and \(\bw\text{,}\) then it is a linear combination of \(\bv\) alone:
    \begin{equation*} \bu=\lambda\bv+\mu\bw=\lambda\bv-2\mu\bv=(\lambda-2\mu)\bv\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 2.4.12.

    Consider the vectors \(\bv=\colvec{1 \\ 2}\) and \(\bu=\colvec{-1 \\ 1}\text{.}\) In this case, \(\vspan(\bv,\bw)=\R^2\text{.}\) In other words, every vector in \(\R^2\) can be written as a linear combination of \(\bv\) and \(\bw\text{.}\) In order to see this, let \(\bb=\colvec{b_1 \\ b_2}\) be a vector and \(\R^2\text{.}\) We want to find real numbers \(x\) and \(y\) such that
    \begin{equation*} x\colvec{1 \\ 2} + y\colvec{-1 \\ 1}=\colvec{b_1 \\ b_2}\text{,} \end{equation*}
    which leads us to consider the linear system
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp - \ampp y \amp = \ampp b_1 \\ 2x \ampp + \ampp y \amp = \ampp b_2 \end{alignedat} \right. \end{equation*}
    You may recall that we saw that if the determinant of a system with two equations and two variables is non-zero, then the system has exactly one solution. The determinant of this system is \(1+2=3\text{,}\) so it always has a single solution.

    In general, how are we to give a nice description of the span of a collection of vectors?
    ðŸ”—
    Letâ€™s say that we have a set of vectors \(S=\{\bv_1,\ldots,\bv_m\}\) in \(\R^n\text{.}\) There are a few operations that we can perform on these vectors that will preserve their span:
    1. When calculating the span, the order in which the generating vectors appear does not matter.
      ðŸ”—
      ðŸ”—
    2. We may replace any vector in \(S\) by a non-zero scalar multiple.
      ðŸ”—
      ðŸ”—
    3. We may add a multiple of a vector in \(S\) to another vector in \(S\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    If you suspect that you are having a dÃ©jÃ  vu, you are right. These properties are analogues of the elementary row operations that we studied in Chapter 1. However, since we represent vectors as columns, we are performing elementary column operations instead!
    ðŸ”—
    If we now consider the matrix whose columns are the vectors \(\bv_1,\bv_2,\ldots,\bv_m\text{,}\) every time we perform an elementary column operation on it, each column will still be an element in \(V=\vspan(\bv_1,\ldots,\bv_m)\) and the set of all columns will still generate \(\vspan(\bv_1,\ldots,\bv_m)\text{.}\) We can therefore use the Gauss-Jordan process on columns to end up with a matrix in reduced column echelon form where the columns generate \(V\text{.}\) Let me show you a few examples that will hopefully convince you that the generating set for \(\vspan(\bv_1,\ldots,\bv_m)\) that we obtain from applying the Gauss-Jordan elimination process to the matrix with columns \(\bv_1,\ldots,\bv_m\) offers a much better description of this space.
    ðŸ”—

    Example 2.4.13.

    We start we the two vectors from ExampleÂ 2.4.12. We want to find a nice description of \(\vspan(\bv,\bw)\) where \(\bv=\colvec{1 \\ 2}\) and \(\bw=\colvec{-1 \\ 1}\text{.}\) Letâ€™s consider the matrix whose columns are these two vectors:
    \begin{equation*} \begin{pmatrix} 1 \amp -1 \\ 2 \amp 1\end{pmatrix}\text{.} \end{equation*}
    The Gauss-Jordan elimination process on columns goes as follows:
    \begin{equation*} \begin{pmatrix} 1 \amp -1 \\ 2 \amp 1\end{pmatrix}\xrightarrow{C_2\to C_2+C_1}\begin{pmatrix} 1\amp 0 \\ 2 \amp 3 \end{pmatrix}\xrightarrow{C_2\to \frac{1}{3}C_2}\begin{pmatrix} 1\amp 0 \\ 2 \amp 1\end{pmatrix}\xrightarrow{C_1\to C_1-2C_2}\begin{pmatrix} 1 \amp 0 \\ 0 \amp 1\end{pmatrix}\text{.} \end{equation*}
    What this means is that \(\vspan(\bv,\bw)\) is precisely the set of vectors in \(\R^2\) generated by the standard unit vectors \(\colvec{1 \\ 0}\) and \(\colvec{0 \\ 1}\text{.}\) This is of course the whole of \(\R^2\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.14.

    Letâ€™s apply the same procedure to understand the span of \(\bv=\colvec{1 \\ 2}\) and \(\bw=\colvec{-2 \\ -4}\text{.}\)
    \begin{equation*} \begin{pmatrix} 1 \amp -2 \\ 2 \amp -4 \end{pmatrix}\xrightarrow{C_2\to C_2+2C_1}\begin{pmatrix} 1 \amp 0 \\ 2 \amp 0\end{pmatrix}\text{.} \end{equation*}
    We conclude that \(\vspan(\bv,\bw)=\vspan(\bv)\text{.}\)

    Example 2.4.15.

    Consider the vectors \(\bv_1=\colvec{2 \\ 0 \\ 2},\bv_2 = \colvec{1 \\ 2 \\ 3}\) and \(\bv_3=\colvec{0 \\ -1 \\ -1}\text{.}\) Letâ€™s try to find a nice description of the span of these three vectors.
    \begin{align*} \begin{pmatrix} 2 \amp 1 \amp 0 \\ 0 \amp 2 \amp -1 \\ 2 \amp 3 \amp -1 \end{pmatrix} \amp \xrightarrow{C_1\to \frac{1}{2}C_1} \begin{pmatrix}1 \amp 1 \amp 0 \\ 0 \amp 2 \amp -1 \\ 1 \amp 3 \amp -1 \end{pmatrix}\xrightarrow{C_2\to C_2-C_1}\begin{pmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 2 \amp -1 \\ 1 \amp 2 \amp -1\end{pmatrix}\\ \amp\xrightarrow{C_2\to\frac{1}{2}C_2} \begin{pmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp -1 \\ 1 \amp 1 \amp -1 \end{pmatrix}\xrightarrow{C_3\to C_3-C_2}\begin{pmatrix}1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 1 \amp 1\amp 0\end{pmatrix}\text{.} \end{align*}
    This means that the span of \(\bv_1,\bv_2,\bv_3\) is the set of vectors in \(\R^3\) of the form
    \begin{equation*} x\colvec{1 \\ 0 \\ 1}+y\colvec{0 \\ 1 \\ 1}=\colvec{x \\ y \\ x+y}\text{,} \end{equation*}
    where \(x\) and \(y\) are real numbers.

    Proposition 2.4.16.

    Let \(S\) be a set of vectors in \(\R^n\text{.}\) There is a finite subset \(T\subseteq S\) that is linearly independent and that satisfies \(\vspan(T)=\vspan(S)\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    The key ingredient is the fact that any set of vectors in \(\R^n\) with more than \(n\) vectors is necessarily linearly dependent. (See ExerciseÂ 2.3.10)
    ðŸ”—
    Define \(V:=\vspan(S)\text{.}\) Start by choosing a maximal subset \(T\) of vectors in \(S\) that is linearly independent (see the exercises). This means that if we add any other vector of \(V\) to \(T\) this set becomes linearly dependent. Since \(T\) is linearly independent, it contains finitely many vectors. (In fact, at most \(n\text{.}\)) We only need to show that \(\vspan(T)=V\text{.}\)
    ðŸ”—
    Of course, \(\vspan(T)\subseteq V\text{.}\) If there were a vector \(\bv\in V\) not in \(\vspan(T)\text{,}\) then \(T\cup\{v\}\) would be linearly independent. But this cannot happen because \(T\) is a maximally linearly independent subset of \(V\text{.}\) This means that \(V\subseteq \vspan(T)\text{.}\) Therefore, \(T\) spans \(V\text{.}\)
    ðŸ”—
    ðŸ”—

    Theorem 2.4.17.

    If \(V\) is a linear subspace of \(\R^n\text{,}\) then there is a finite linearly independent set \(S\) of vectors in \(V\) such that \(\vspan(S)=V\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    This is a direct consequence of PropositionÂ 2.4.16. Indeed, the set \(V\) itself spans \(V\text{.}\) Using PropositionÂ 2.4.16, we immediately conclude that there is a finite subset of \(V\) that spans \(V\) and that is linearly independent.

    Definition 2.4.18.

    A linearly independent set \(S\) of a linear subspace \(V\) of \(\R^n\) that spans \(V\) is called a basis of \(V\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.19.

    The set of standard unit vectors \(\be_1,\be_2,\ldots,\be_n\) in \(\R^n\) is a basis of \(\R^n\) called the standard basis of \(\R^n\text{.}\) In order to show this, we need to show two things: first, we need to show that the set of standard unit vectors is linearly independent; after this, we need to show that this set spans \(\R^n\text{.}\)
    ðŸ”—
    We already saw that the set of standard of unit vectors is linearly independent, but letâ€™s see why this is the case again. Denote by \(e_{ij}\) the \(j\)th coordinate of \(\be_i\text{.}\) By definition, we have \(e_{ij}=0\) if \(i\neq j\) and \(e_{ii}=1\text{.}\) If \(\lambda_1,\lambda_2,\ldots,\lambda_n\in\R\) are such that
    \begin{equation*} \lambda_1\be_1+\lambda_2\be_2+\ldots+\lambda_n\be_n=\bz, \end{equation*}
    then we have \(\lambda_1 e_{1j}+\lambda_2 e_{2j}+\ldots+\lambda_n e_{nj}=0\) for every \(j\text{.}\) But this means that \(\lambda_j=0\) for every \(j\text{.}\) Therefore, the set of standard unit vectors is linearly independent.
    ðŸ”—
    Now consider a vector \(\bv\in\R^n\text{.}\) If \(x_i\) denotes the \(i\)th entry of \(\bv\text{,}\) then it is easy to see that \(\bv=x_1\be_1 +x_2\be_2+\ldots+x_n\be_n\text{,}\) which means that \(\be_1,\be_2,\ldots,\be_n\) spans \(\R^n\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.20.

    ExampleÂ 2.4.19 says, in particular, that the standard basis of \(\R^2\) is the set formed by the vectors \(\colvec{1 \\ 0}\) and \(\colvec{0 \\ 1}\text{.}\) But there are many more bases of \(\R^2\text{.}\) (In fact, there are infinitely many: any two non-collinear vectors are a basis of \(\R^2\text{.}\)) For example, the vectors \(\bv=\colvec{-1 \\ 1}\) and \(\bu=\colvec{2 \\ 1}\) are a basis of \(\R^2\text{.}\) To see this, we first need to show that they are linearly independent, and then that they span \(\R^2\text{.}\)
    ðŸ”—
    If \(a,b\in\R\) are scalars such that \(a\bv+b\bu=\bz\text{,}\) then
    \begin{equation*} \colvec{-a \\ a}+\colvec{2b \\ b}=\colvec{0 \\ 0}\text{,} \end{equation*}
    and so \(2b-a=0\) and \(b+a=0\text{.}\) It can be quickly checked that the only solution to this system is \(a=0\) and \(b=0\text{.}\) This means that \(\bv\) and \(\bu\) are linearly independent.
    ðŸ”—
    Now we need to show that every vector in \(\R^2\) can be expressed as a linear combination of \(\bv\) and \(\bu\text{.}\) Since we know that \(\colvec{1 \\ 0}\) and \(\colvec{0 \\ 1}\) are a basis of \(\R^2\text{,}\) it will suffice to show that we can express \(\colvec{1 \\ 0}\) and \(\colvec{0 \\ 1}\) as linear combinations of \(\bv\) and \(\bu\text{.}\) (Pause here and think why this is true.) Letâ€™s show that we can express \(\colvec{1 \\ 0}\) as a linear combination of \(\bv\) and \(\bv\text{.}\) The case of \(\colvec{0 \\ 1}\) is almost identical.
    ðŸ”—
    We want to find scalars \(a,b\in \R\) such that \(a\bv+b\bu=\be_1\text{.}\) This means solving the equation
    \begin{equation*} a\colvec{-1 \\ 1} + b\colvec{2 \\ 1}=\colvec{1 \\ 0}\text{,} \end{equation*}
    which is equivalent to the system
    \begin{equation*} \left\{ \begin{alignedat}{5} -a \ampp + \ampp 2b \amp = \ampp 1 \\ a \ampp + \ampp b \amp = \ampp 0 \end{alignedat} \right. \end{equation*}
    Solving the system for \(a\) and \(b\text{,}\) we find that the system has the solution \(a=-1/3\) and \(b=1/3\text{.}\)
    ðŸ”—
    ðŸ”—
    The obvious question now is that of finding a basis for a given vector space. If we are given a finite set of vectors \(S\) and the vector space in question in \(\vspan(S)\text{,}\) then we already know how to do that: this is precisely what we were doing above when we used elementary column operations to find a nice description of \(\vspan(S)\text{!}\) However, vector spaces may not be described using a given generator set. Instead, they may be given by a set of linear equations.
    ðŸ”—

    Example 2.4.21.

    Consider the subset \(V\) of vectors \((x,y,z)\) of \(\R^3\) that satisfy \(x+2y-z=0\text{.}\) This is a linear subspace of \(\R^3\text{.}\) This equation has infinitely many solutions that can be given in parametric form by \((t-2s, s, t)\) where \(s,t\in\R\text{.}\) If \(s=0\) and \(t=1\text{,}\) we obtain the vector \(\colvec{1 \\ 0 \\ 1}\text{.}\) If we now set \(s=1\) and \(t=0\text{,}\) we get \(\colvec{-2 \\ 1\\ 0}\text{.}\) It is easy to see that these two vectors are linearly independent and that they span all vectors of the form \((t-2s, s, t)\) because
    \begin{equation*} \colvec{t-2s \\ s \\ t} = t\colvec{1 \\ 0 \\ 1} + s\colvec{-2 \\ 1 \\ 0}\text{.} \end{equation*}
    Therefore, the vectors \(\colvec{1 \\ 0 \\ 1}\) and \(\colvec{-2 \\ 1 \\0}\) form a basis of \(V\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.22.

    Let \(V\) be the linear subspace of \(\R^4\) consisting of vectors \((x,y,z,w)\) in \(\R^4\) satisfying the equations \(2x-3z=0\) and \(2y+z+w=0\text{.}\) Solving the system, we see that \(z\) and \(w\) are free variables and the solutions to the system can be given in parametric form as \(\left(\frac{3}{2}s, -\frac{1}{2}s-\frac{1}{2}t, s, t\right)\text{,}\) where \(s,t\in\R\text{.}\) Setting \(s=0\) and \(t=2\text{,}\) we find the solution \(\colvec{0 \\ -1 \\ 0 \\ 2}\text{.}\) If we now set \(s=2\) and \(t=0\text{,}\) we get \(\colvec{3 \\ -1 \\ 2 \\ 0}\text{.}\) Since these two vectors are linearly independent and
    \begin{equation*} \colvec{\frac{3}{2}s \\ -\frac{1}{2}s-\frac{1}{2}t \\ s \\ t} = \frac{s}{2}\colvec{3 \\ -1 \\ 2 \\ 0}+\frac{t}{2}\colvec{0 \\ -1 \\ 0 \\ 2}\text{,} \end{equation*}
    they constitute a basis of \(V\text{.}\)
    ðŸ”—
    ðŸ”—
    So every linear subspace of \(\R^n\) has a basis. While there are infinitely many basis for a given non-zero linear subspace of \(\R^n\text{,}\) they will all have the same size.

    Lemma 2.4.23.

    Let \(V\) be a linear subspace of \(\R^n\text{.}\) If \(B\) is a basis of \(V\) and \(S\) is a subset of vectors of \(V\) whose size is strictly larger than the size of \(B\text{,}\) then \(S\) is linearly dependent.
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(n\) be the size of \(B\) and \(m\) be the size of \(S\text{.}\) By hypothesis, \(m\gt n\text{.}\) Let \(B=\{\bv_1,\ldots,\bv_n\}\) and \(S=\{\bu_1,\ldots,\bu_m\}\text{.}\) Since \(B\) is a basis, every vector in \(S\) can be written as a linear combination of vectors in \(B\text{.}\) This means that for each \(1\leq j\leq m\text{,}\) there are scalars \(a_{1j}, a_{2j},\ldots, a_{nj}\) such that
    \begin{equation*} a_{1j}\bv_1+a_{2j}\bv_2+\ldots + a_{nj}\bv_n=\bu_j \end{equation*}
    for every \(j\) in \(\{1,\ldots,m\}\text{.}\) The aim is to use this to find a non-trivial linear combination of the \(\bu_i\)â€™s that represents the zero vector.
    ðŸ”—
    To be clear, we are trying to find scalars \(x_1,x_2,\ldots,x_m\text{,}\) not all zero, such that \(x_1\bu_1+x_2\bu_2+\ldots+x_n\bu_n=\bz\text{.}\) It will be enough that these scalars satisfy
    \begin{equation*} x_1a_{i1} + x_2a_{i2}+\ldots+x_ma_{im}=0\text{.} \end{equation*}
    (You should try to figure out why this is true.) In other words, we need to solve a homogeneous linear system with \(n\) equations and \(m\) variables. Since \(m\gt n\text{,}\) there are infinitely many solutions, which means that there are non-zero solutions too.
    ðŸ”—
    ðŸ”—

    Theorem 2.4.24.

    Let \(V\) be a linear subspace of \(\R^n\text{.}\) Every basis of \(V\) has the same size.
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose, for the sake of contradiction, that we have two basis \(B_1\) and \(B_2\) of different sizes. Say that the size of \(B_1\) is \(n\) and that of \(B_2\) is \(m\text{.}\) We may assume, without loss of generality, that \(m\gt n\text{.}\) In that case, LemmaÂ 2.4.23 implies that \(B_2\) is linearly dependent, which contradicts the fact that it is a basis. Therefore, \(B_1\) and \(B_2\) must have the same size.
    ðŸ”—
    ðŸ”—

    Definition 2.4.25.

    Let \(V\) be a linear subspace of \(\R^n\text{.}\) The size of any basis of \(V\) is called the dimension of \(V\text{.}\)
    ðŸ”—
    ðŸ”—
    This definition allows us to restate LemmaÂ 2.4.23. Indeed, we may simply say that any subset of \(V\) of size \(\gt \dim(V)\) is linearly dependent.

    Example 2.4.26.

    Consider the linear subspace \(V\) of \(\R^4\) defined as the set of solutions to the equation \(x-2y+3z-w=0\text{.}\) The variables \(y,z\) and \(w\) are free variables. The set of solutions to this equation in \(\R^4\) can be given in parametric form by \((2s-3t+r,s,t,r)\text{,}\) where \(s,t,r\in\R\text{.}\) It is easy to see that the vectors \(\bv = \colvec{2 \\ 1 \\ 0 \\ 0}\text{,}\) \(\bu=\colvec{-3 \\ 0 \\ 1 \\ 0}\) and \(\bw = \colvec{1 \\ 0 \\ 0 \\ 1}\) are linearly independent and that
    \begin{equation*} \colvec{2s-3t+r \\ s \\ t \\ r} = s\bv + t\bu+r\bw \end{equation*}
    for any \(s,t,u\in\R\text{.}\) This means that \(\bv, \bu\) and \(\bw\) form a basis of \(V\text{.}\) Therefore, \(\dim(V)=3\text{.}\)

    ðŸ”—
    Note that the number of free variables in a linear system always matches the dimension of the solution set of that linear system.

    If one linear subspace is contained in another, then, as you might expect, the dimension of the one that contains is at least as big as that of the one that is contained. This is the content of the following proposition. Try to prove it using LemmaÂ 2.4.23.

    Proposition 2.4.27.

    Let \(V\) and \(W\) be two linear subspaces of \(\R^n\text{.}\) If \(V\subseteq W\text{,}\) then \(\dim(V)\leq \dim(W)\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose, for the sake of contradiction, that there are linear subspaces \(V\) and \(W\) of \(\R^n\) such that \(V\subseteq W\) and \(\dim(V)\gt \dim(W)\text{.}\) Let \(B\) be a basis of \(V\text{.}\) Since \(V\) is contained in \(W\text{,}\) the set \(B\) is contained in \(W\) as well. However, \(\#B=\dim(V)\gt \dim(W)\text{,}\) which, according to LemmaÂ 2.4.23, means that \(B\) is linearly dependent. This, however, contradicts the fact that it is a basis of \(V\text{.}\) It follows that we cannot have \(\dim(V) \gt \dim(W)\text{,}\) and so it must be the case that \(\dim(V)\leq \dim(W)\text{.}\)

    ðŸ”—

    Proposition 2.4.28.

    Let \(V\) be a linear subspace of \(\R^n\) of dimension \(d\text{.}\) Any linearly independent set of vectors in \(V\) of size \(d\) is a basis of \(V\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(S\) be a subset of vectors of \(V\) of size \(d\) that is linearly independent. We only need to show that every vector of \(V\) can be written as a linear combination of vectors in \(S\text{.}\)
    ðŸ”—
    Choose a vector \(v\in V\text{.}\) We know from LemmaÂ 2.4.23 that any set of vectors in \(V\) whose size is larger than \(d\) is linearly dependent. This means that \(S\cup\{d\}\) must be linearly dependent because its size is \(d+1\text{.}\) Using LemmaÂ 2.3.15, we conclude that the vector \(v\) can be written as a linear combination of vectors in \(S\text{.}\) This shows that \(S\) is a basis of \(V\text{.}\)

    Let \(V\) and \(W\) be linear subspaces of \(\R^n\) such that \(V\subseteq W\text{.}\) If \(\dim(V)=\dim(W)\text{,}\) then \(V=W\text{.}\)

    Example 2.4.30.

    Letâ€™s use the notions of basis and dimension to describe all the linear subspaces of \(\R^2\text{.}\)
    ðŸ”—
    As we know, \(\R^2\) has a standard basis formed by the standard unit vectors \(\be_1\) and \(\be_2\text{.}\) This means that \(\dim(\R^2)=2\text{.}\) (In general, \(\dim(\R^n)=n\) because the standard basis of \(\R^n\) consists of the \(n\) standard unit vectors \(\be_1,\be_2,\ldots,\be_n\text{.}\)) If \(V\subseteq \R^2\) is a linear subspace of \(\R^2\text{,}\) then we know from PropositionÂ 2.4.27 that \(\dim(V)\leq 2\text{.}\) Therefore, either \(\dim(V)=0\text{,}\) or \(\dim(V)=1\text{,}\) or \(\dim(V)=2\text{.}\)
    ðŸ”—
    If \(\dim(V)=2\text{,}\) then, by CorollaryÂ 2.4.29, we must have \(V=\R^2\text{.}\)
    ðŸ”—
    If \(\dim(V)=0\text{,}\) then this means that \(V\) is the span of the empty set (the only set of size \(0\)). The span of the empty set is the linear subspace of \(\R^2\) consisting of the origin only.
    ðŸ”—
    Finally, the most interesting case: \(\dim(V)=1\text{.}\) In this case, the vector space \(V\) is generated by a single vector \(\bv\text{.}\) If \(\bv=(a,b)\text{,}\) this means that we are looking at the subset of \(\R^2\) consisting of points of the form \((\lambda a, \lambda b)\) for some \(\lambda\in \R\text{.}\) In other words, we are looking at the line through the origin that is generated by the vector \(\bv\text{.}\) (This is the line with equation \(bx-ay=0\text{.}\)) So, every \(1\)-dimensional linear subspace of \(\R^2\) is a line passing through the origin. Conversely, a line in \(\R^2\) passing through the origin is always a \(1\)-dimensional linear subspace of \(\R^2\) because it can always be described by a homogeneous linear equation of the form \(ax+by=0\text{.}\)

    The final topic that I want to touch on before moving on to the next section is that of the possibility of extending a linearly independent set of vectors in a given linear subspace of \(\R^n\) to a basis. This is always possible.
    ðŸ”—

    Theorem 2.4.31.

    Let \(V\) be a linear subspace of \(\R^n\text{,}\) and let \(S\) be a linearly independent set of vectors in \(V\text{.}\) There exists a basis \(B\) of \(V\) such that \(S\subseteq B\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    If the size of \(S\) equals the dimension of \(V\text{,}\) then \(S\) is itself a basis of \(V\) and we are done. If, however, the size of \(S\) is strictly smaller than \(\dim(V)\) we can always extend \(S\) by adding a vector of \(V\) while guaranteeing linear independence. Indeed, if \(\bv\) is a vector of \(V\) not in \(\vspan(S)\text{,}\) then \(S\cup\{\bv\}\) is strictly larger than \(S\) and it must be a linearly independent set of vectors in \(V\text{,}\) for otherwise \(\bv\) would be in the span of \(S\) (see LemmaÂ 2.3.15). Proceeding in this manner, we eventually end up with a linearly independent set \(B\) of size \(\dim(V)\) which contains \(S\text{.}\) This set must be a basis of \(V\text{.}\)
