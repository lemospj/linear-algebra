    Section 2.4 Linear subspaces

    The notion of a linear subspace of a vector space is a central notion in linear algebra.

    Definition 2.4.1.

    A subset \(V\) of \(\R^n\) is called a linear subspace of \(\R^n\) if the following conditions hold:
    1. If \(\bv,\bu\in V\text{,}\) then \(\bv+\bu\in V\) (i.e., \(V\) is closed under addition).

    2. If \(\lambda\in R\) and \(\bv\in V\text{,}\) then \(\lambda\bv\in \R\) (i.e., \(V\) is closed under scalar multiplication).

    Any set of vectors in \(\R^n\) generates a linear subspace called its span.
    ðŸ”—

    Definition 2.4.3.

    Let \(S\) be a set of vectors in \(\R^n\text{.}\) The span of \(S\text{,}\) denoted \(\vspan(S)\text{,}\) is the set of vectors in \(\R^n\) that can be written as a linear combination of elements in \(S\text{.}\) We say that the set \(S\) generates \(\vspan(S)\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 2.4.4.

    Consider the vector \(\bv=\colvec{1 \\ 2}\) in \(\R^2\text{.}\) All linear combinations of the set formed by this single vector are of the form \(\lambda\bv\) for some scalar \(\lambda\in \R\text{.}\) Therefore, \(\vspan(\bv)\) is simply the line in \(\R^2\) that passes through the origin and has the same direction as the vector \(\bv\text{.}\)
    ðŸ”—
    Consider now the set \(S\) formed by the vectors \(\bv=\colvec{1 \\ 2}\) and \(\bw=\colvec{-2 \\ -4}\text{.}\) Since \(\bw=-2\bv\) the span \(\vspan(\bv,\bw)\) is the same as \(\vspan(\bv)\text{.}\) Indeed, if a vector \(\bu\in\R^2\) is a linear combination of \(\bv\) and \(\bw\text{,}\) then it is a linear combination of \(\bv\) alone:
    \begin{equation*} \bu=\lambda\bv+\mu\bw=\lambda\bv-2\mu\bv=(\lambda-2\mu)\bv\text{.} \end{equation*}

    Example 2.4.7.

    Consider the vectors \(\bv=\colvec{1 \\ 2}\) and \(\bu=\colvec{-1 \\ 1}\text{.}\) In this case, \(\vspan(\bv,\bw)=\R^2\text{.}\) In other words, every vector in \(\R^2\) can be written as a linear combination of \(\bv\) and \(\bw\text{.}\) In order to see this, let \(\bb=\colvec{b_1 \\ b_2}\) be a vector and \(\R^2\text{.}\) We want to find real numbers \(x\) and \(y\) such that
    \begin{equation*} x\colvec{1 \\ 2} + y\colvec{-1 \\ 1}=\colvec{b_1 \\ b_2}\text{,} \end{equation*}
    which leads us to consider the linear system
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp - \ampp y \amp = \ampp b_1 \\ 2x \ampp + \ampp y \amp = \ampp b_2 \end{alignedat} \right. \end{equation*}
    You may recall that we saw that if the determinant of a system with two equations and two variables is non-zero, then the system has exactly one solution. The determinant of this system is \(1+2=3\text{,}\) so it always has a single solution.

    If we now consider the matrix whose columns are the vectors \(\bv_1,\bv_2,\ldots,\bv_m\text{,}\) every time we perform an elementary column operation on it, each column will still be an element in \(V=\vspan(\bv_1,\ldots,\bv_m)\) and the set of all columns will still generate \(\vspan(\bv_1,\ldots,\bv_m)\text{.}\) We can therefore use the Gauss-Jordan process on columns to end up with a matrix in reduced column echelon form where the columns generate \(V\text{.}\) Let me show you a few examples that will hopefully convince you that the generating set for \(\vspan(\bv_1,\ldots,\bv_m)\) that we obtain from applying the Gauss-Jordan elimination process to the matrix with columns \(\bv_1,\ldots,\bv_m\) offers a much better description of this space.

    Example 2.4.8.

    We start we the two vectors from ExampleÂ 2.4.7. We want to find a nice description of \(\vspan(\bv,\bw)\) where \(\bv=\colvec{1 \\ 2}\) and \(\bw=\colvec{-1 \\ 1}\text{.}\) Letâ€™s consider the matrix whose columns are these two vectors:
    \begin{equation*} \begin{pmatrix} 1 \amp -1 \\ 2 \amp 1\end{pmatrix}\text{.} \end{equation*}
    The Gauss-Jordan elimination process on columns goes as follows:
    \begin{equation*} \begin{pmatrix} 1 \amp -1 \\ 2 \amp 1\end{pmatrix}\xrightarrow{C_2\to C_2+C_1}\begin{pmatrix} 1\amp 0 \\ 2 \amp 3 \end{pmatrix}\xrightarrow{C_2\to \frac{1}{3}C_2}\begin{pmatrix} 1\amp 0 \\ 2 \amp 1\end{pmatrix}\xrightarrow{C_1\to C_1-2C_2}\begin{pmatrix} 1 \amp 0 \\ 0 \amp 1\end{pmatrix}\text{.} \end{equation*}
    What this means is that \(\vspan(\bv,\bw)\) is precisely the set of vectors in \(\R^2\) generated by the standard unit vectors \(\colvec{1 \\ 0}\) and \(\colvec{0 \\ 1}\text{.}\) This is of course the whole of \(\R^2\text{.}\)
