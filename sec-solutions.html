    Section 1.1 Solutions to systems of linear equations

    I will start by defining what I mean by a linear equation and a system of linear equations. When mathematicians use a mathematical concept, they always mean something very precise. Humpty Dumpty, the character in Lewis Carrollâ€™s Through the Looking Glass famously said, "When I use a word,... it means just what I choose it to mean --- neither more nor less." Humpty Dumpty must have been a mathematician. (Lewis Carroll certainly was!) For this reason, it is very important to clearly define a concept before using it, and anyone learning mathematics at any level must pay close attention to what a definition says and to what it does not say. The best way to understand a definition is to find and come up with examples and counter-examples. You should train yourself to always want examples after a definition. If the text that you are reading does not provide one, come up with examples and counter-examples yourself!
    ðŸ”—

    Definition 1.1.1.

    A linear equation in the variables \(x_1,\ldots, x_n\) is an equation that can be written in the form
    \begin{equation*} a_1x_1+a_2x_2+\ldots+a_nx_n=b \end{equation*}
    where \(b,a_1,\ldots,a_n\in\R\) are real constants. The real numbers \(b,a_1,\ldots,a_n\) are called the coefficients of the linear equation. If \(b=0\text{,}\) we say that the linear equation is homogeneous.
    ðŸ”—
    ðŸ”—

    Example 1.1.2.

    The equation \(5x+4y=2\) is a linear equation in the variables \(x,y\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.1.3.

    The equation \(\sqrt{2}x-\pi y=ez\) is a linear equation in the variables \(x,y,z\) because we can rewrite it in the form \(\sqrt{2}x-\pi y - ez=0\text{.}\) In fact, it is a homogeneous linear equation.
    ðŸ”—
    ðŸ”—

    Example 1.1.4.

    On the other hand, the equation \(2\sqrt{x}+5y=1\) is not linear because it involves the non-linear term \(\sqrt{x}\text{.}\) For a similar reason, the equation \(x^2=y^2-1\) is not linear either.
    ðŸ”—
    ðŸ”—

    Example 1.1.5.

    Even though \(4x-3z+1=1\) may not seem to be a homogeneous linear equation at first sight, note that once we subtract \(1\) from both sides of the equation we end up with \(4x-3z=0\text{.}\) Therefore, this is a homogeneous linear equation.
    ðŸ”—
    ðŸ”—

    Definition 1.1.6.

    A system of linear equations (also called a linear system) is a finite set of linear equations. The variables occurring in the system are called the unknowns. If all the linear equations in this system are homogeneous, we say that the system itself is homogeneous. Otherwise, we say that the system is called inhomogeneous.
    ðŸ”—
    ðŸ”—
    When we write down a system of linear equations, we typically write each of the finitely many equations forming it in a line of their own. For example,
    \begin{equation} \left\{ \begin{alignedat}{5} 3x \amp\amp\; + \amp\amp\; 2y \amp = \amp\amp\; -7\\ 2y \amp\amp\; - \amp\amp\; z \amp = \amp\amp\; 0\\ \amp\amp \amp\amp w \amp = \amp\amp\; 0 \end{alignedat} \right.\tag{1.1.1} \end{equation}
    is a non-homogeneous system of linear equation with unknowns \(x,y,z,w\text{.}\) It consists of three linear equations: \(3x+2y=-7\text{,}\) \(2y-z=0\) and \(w=0\text{.}\)
    ðŸ”—
    In general, a system with \(m\) equations and \(n\) unknowns \(x_1,\ldots,x_n\) can be written in the form
    \begin{equation*} \left\{ \begin{alignedat}{9} \amp a_{11}x_1 \amp\amp+ \amp\amp a_{12}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{1n}x_n \amp\amp = \amp\amp \; b_1\\ \amp a_{21}x_1 \amp\amp + \amp\amp a_{22}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp \; a_{2n}x_n \amp\amp = \amp\amp \; b_2\\ \amp \quad\vdots \quad \amp\amp \quad \amp\amp\quad\vdots \quad \amp\amp \amp\amp\quad \ldots \quad \amp\amp \amp\amp\quad\vdots \quad \amp\amp \amp\amp\; \vdots\\ \amp a_{m1}x_1 \amp\amp + \amp\amp a_{m2}x_2 \amp\amp + \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{mn}x_n \amp\amp = \amp\amp \; b_m \end{alignedat}\right. \end{equation*}
    This way of writing a linear system is extremely helpful. Each column to the left-hand side of the equal sign corresponds to an unknown. More precisely, the \(k\)th column corresponds to the variable \(x_k\text{.}\) The coefficients of the system are the constants \(a_{ij}\text{,}\) where \(1\leq i\leq m\) and \(1\leq j\leq n\text{.}\) Note that the pair \((i,j)\) gives you the position of \(a_{ij}\) in the linear system above: the coefficient \(a_{ij}\) is located in the \(i\)th row of the \(j\)th column.
    ðŸ”—
    The linear system (1.1.1) can be written in the form above by introducing zeros whenever a variable is not present in one of the linear equations forming the system:
    \begin{equation} \left\{ \begin{alignedat}{9} 3x \amp\amp\; + \amp\amp\; 2y \amp\amp\; +\amp\amp\; 0z \amp\amp\; + \amp\amp\; 0w \amp = \amp\amp\; -7\\ 0x \amp\amp\; + \amp\amp\; 2y \amp\amp\; - \amp\amp\; z \amp\amp\; + \amp\amp\; 0w \amp = \amp\amp\; 0\\ 0x \amp\amp\; + \amp\amp\; 0y \amp\amp\; + \amp\amp\; 0z \amp\amp\; + \amp\amp\; w \amp = \amp\amp\; 0 \end{alignedat} \right.\tag{1.1.2} \end{equation}
    ðŸ”—

    Definition 1.1.7.

    A solution to a linear system
    \begin{equation*} \left\{ \begin{alignedat}{9} \amp a_{11}x_1 \amp\amp+ \amp\amp a_{12}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{1n}x_n \amp\amp = \amp\amp \; b_1\\ \amp a_{21}x_1 \amp\amp + \amp\amp a_{22}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp \; a_{2n}x_n \amp\amp = \amp\amp \; b_2\\ \amp \quad\vdots \quad \amp\amp \quad \amp\amp\quad\vdots \quad \amp\amp \amp\amp\quad \ldots \quad \amp\amp \amp\amp\quad\vdots \quad \amp\amp \amp\amp\; \vdots\\ \amp a_{m1}x_1 \amp\amp + \amp\amp a_{m2}x_2 \amp\amp + \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{mn}x_n \amp\amp = \amp\amp \; b_m \end{alignedat}\right. \end{equation*}
    in the variables \(x_1,\ldots,x_n\) is an \(n\)-tuple \((s_1,\ldots,s_n)\in\R^n\) that is a solution to every linear equation in the system. This means that
    \begin{equation*} a_{r1}s_1+a_{r2}s_2+\ldots+a_{rn}s_n=b_r \end{equation*}
    for every \(r\) in the set \(\{1,\ldots,m\}\text{.}\) The collection of solutions to a linear system is called its solution set.
    ðŸ”—
    ðŸ”—

    Definition 1.1.8.

    A linear system is said to be consistent if it has at least one solution. A linear system without solutions is called inconsistent.
    ðŸ”—
    ðŸ”—
    Much of what we will be doing in this chapter is learning ways of transforming linear systems into simpler linear systems while preserving the solution set of the initial system.
    ðŸ”—

    Definition 1.1.9.

    Two linear systems are said to be equivalent if they have the same solution set.
    ðŸ”—
    ðŸ”—

    Example 1.1.10.

    Letâ€™s consider the case of a linear equation in two variables. (If you have a look at the definition of linear systems, you will notice that I never required a linear system to have more than one equation!) For instance, we can consider the equation
    \begin{equation*} 2x-3y=1\text{.} \end{equation*}
    There are infinitely many solutions to this equation. Indeed, solving for the equation for \(y\text{,}\) we get
    \begin{equation*} y=\frac{2}{3}x-\frac{1}{3}\text{.} \end{equation*}
    We may now freely choose a real value \(t\) for \(x\text{,}\) and for each such value,
    \begin{equation} x=t,\quad y=\frac{2}{3}t-\frac{1}{3}\qquad (t\in\R)\tag{1.1.3} \end{equation}
    is a solution to the equation. For example, you could choose \(x=3\text{,}\) and this would force \(y\) to be \(\frac{5}{3}\text{,}\) meaning that \((3,5/3)\) is a solution to the linear equation. Indeed, \(2\times 3 - 3\times \frac{5}{3}=1\text{.}\) Therefore, equation (1.1.3) is a recipe to cook up solutions to our linear equation. When there are infinitely many solutions to linear equations, a recipe like this is the best we can hope for. We call this a parametric solution. In this case, the solution has a single parameter \(t\text{.}\)
    ðŸ”—
    When working with systems of at most three variables, we can attempt to visualize the solution sets of our linear systems. I would strongly encourage you to do this as often as possible. This practice will greatly improve your geometric thinking and build your intuition, both of which are invaluable assets in mathematics.
    ðŸ”—
    Letâ€™s plot the solutions to the equation \(2x-3y=1\) of our previous example.
    ðŸ”—
    ðŸ”—
    There are a few important operations that we can perform on linear systems that preserve the solution set, i.e. that turn one linear system into an equivalent one. We will be using these operations systematically to simplify any given system. These operations are:
    1. Multiplication of one of the equations by a non-zero constant.
      ðŸ”—
      ðŸ”—
    2. Equation swapping.
      ðŸ”—
      ðŸ”—
    3. Addition of a constant multiple of one equation to another equation in the system.
      ðŸ”—
      ðŸ”—
    These three operations are known as elementary row operations.
    ðŸ”—

    Example 1.1.12.

    Consider the following linear system:
    \begin{equation} \left\{ \begin{alignedat}{5} x \amp\amp\; + \amp\amp\; y \amp = \amp\amp\; -1\\ 2x \amp\amp\; - \amp\amp\; 3y \amp = \amp\amp\; 1 \end{alignedat} \right.\tag{1.1.4} \end{equation}
    Letâ€™s find all the solutions to this linear system. You may feel tempted to solve the second equation for \(y\) or \(x\) and then substitute in the first. That will work. However, for larger systems, that turns out to be a slow and cumbersome method. Since one of our main aims in this first chapter is to find an efficient method to systematically solve systems of linear equations, I will use this particular example to explain how one can use the elementary row operations described above to simplify and solve a linear system. The idea is to eliminate one of the variables of the first equation by adding a suitable multiple of the second equation. An important aspect of this method is that we never move variables to the right-hand side. Letâ€™s remove the term \(2x\) from the second equation. In order to do that, we will subtract \(2\) times the first equation from the second. We get the equivalent linear system
    \begin{equation*} \left\{ \begin{alignedat}{5} x \amp\amp\; + \amp\amp\; y \amp = \amp\amp\; -1\\ \amp\amp -\amp\amp\; 5y \amp = \amp\amp\; 3 \end{alignedat} \right. \end{equation*}
    We can now multiply the second equation by \(-1/5\text{.}\) This gives us
    \begin{equation*} \left\{ \begin{alignedat}{5} x \amp\amp\; + \amp\amp\; y \amp = \amp\amp\; -1\\ \amp\amp \amp\amp y \amp = \amp\amp\; -\frac{3}{5} \end{alignedat} \right. \end{equation*}
    Finally, if we subtract the second equation from the first we obtain
    \begin{equation*} \left\{ \begin{aligned} x \amp = -\frac{2}{5}\\ y \amp = -\frac{3}{5} \end{aligned} \right. \end{equation*}
    Therefore, the solution to our linear system is \((x,y)=(-2/5,-3/5)\text{.}\)
    ðŸ”—
    Letâ€™s plot both equations of the linear system above in the same Cartesian plane to have a geometrical picture of the situation. The solution set of \(2x-3y=1\) is represented by the blue line in the picture above. The solution set of \(x+y=-1\) is represented by the orange line. These two lines intersect at precisely one point. That point is necessarily a common solution to these two linear equations. In other words, it is the only solution to the linear system (1.1.4), which we already know to be the point \((-2/5,-3/5)\text{.}\)
    ðŸ”—
    ðŸ”—
    You can start to see why geometry may be useful when finding solutions to linear systems. In a system with two equations and two unknowns, three situations may occur:
    1. The lines corresponding to the solution sets of the two equations are not parallel, and so they meet at exactly one point. In this situation, there is exactly one solution to the system.
      ðŸ”—
      ðŸ”—
    2. The lines corresponding to the solution sets of the two equations are parallel and distinct, and so they never meet. In this case, there are no solutions to the system.
      ðŸ”—
      ðŸ”—
    3. The lines corresponding to the solution sets of the two equations are the same line. In this case, there are infinitely many solutions.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    A good exercise is to use this insight to prove the following theorem.
    ðŸ”—

    Theorem 1.1.14.

    Consider the linear system in the unknowns \(x,y\) given by
    \begin{equation*} \left\{ \begin{aligned} ax+by \amp = b_1\\ cx+dy \amp = b_2 \end{aligned} \right. \end{equation*}
    where \(a,b,c,d,b_1,b_2\in \R\) are constants. If \(ad-bc\neq 0\text{,}\) then this system has precisely one solution. If, otherwise, \(ad-bc=0\text{,}\) then this system has either none or infinitely many solutions.
    ðŸ”—
    ðŸ”—

    Proof.

    Letâ€™s first treat the case where \(ad-bc\neq 0\text{.}\) In this case, either \(b\neq 0\) or \(d\neq 0\text{.}\) We may assume without loss of generality that \(b\neq 0\text{,}\) because we can always swap the two equations and rename the coefficients. Then the line \(\ell_1\) corresponding to the solution set of the first equation has slope \(-a/b\text{.}\)
    ðŸ”—
    Since we are assuming that \(ad-bc\neq 0\text{,}\) it must also be the case that \(c\neq 0\) or \(d\neq 0\text{.}\) It follows that the solution set \(\ell_2\) of \(cx+dy=b_2\) is either a vertical line, or a line with slope \(-c/d\text{.}\) All we need to do now is to show that the lines \(\ell_1\) and \(\ell_2\) intersect at a single point. It will suffice to show that they are not parallel.
    ðŸ”—
    If \(\ell_2\) is a vertical line, then it is clear that \(\ell_1\) and \(\ell_2\) are not parallel because \(\ell_1\) is not vertical. The other situation is that where \(d\neq 0\) and \(\ell_2\) has slope \(-c/d\text{.}\) If the lines \(\ell_1\) and \(\ell_2\) were parallel, then their slopes would match:
    \begin{equation*} -\frac{a}{b}=-\frac{c}{d}\text{.} \end{equation*}
    But this is equivalent to \(ad-bc=0\text{,}\) which contradicts our assumption. Therefore, \(\ell_1\) and \(\ell_2\) cannot be parallel and therefore meet at exactly one point.
    ðŸ”—
    Letâ€™s now assume that \(ad-bc=0\text{.}\) First, we should deal with two degenerate situations. The first occurs when \(a=b=c=d=0\text{.}\) If both \(b_1\) and \(b_2\) are also \(0\text{,}\) then the system has infinitely many solutions. But if one of the \(b_i\) is not \(0\text{,}\) then the system is inconsistent.
    ðŸ”—
    The second degenerate situation occurs when either \(a=b=0\) or \(c=d=0\text{,}\) but not both. Say that \(a=d=0\) (the other case is dealt with in a similar manner). If \(b_1=0\text{,}\) the first equation is then \(0=0\) and imposes no conditions. The solution set will therefore be the line defined by the second linear equation, yielding infinitely many solutions. On the other hand, if \(b_1\neq 0\text{,}\) there are no solutions.
    ðŸ”—
    Finally, the non-degenerate case is that where the solution set of each linear equation is a line. Letâ€™s first assume that \(b=0\text{.}\) Then the solution set \(\ell_1\) of the first equation is a vertical line. Since \(ad-bc=0\text{,}\) we must have \(d=0\) as well, and the solution set \(\ell_2\) of the second equation must also be a vertical line. They are either parallel (no solutions), or they are exactly the same line (infinitely many solutions).
    ðŸ”—
    If \(b\neq 0\text{,}\) then the slope of \(\ell_1\) is \(-a/b\text{.}\) Since \(ad-bc=0\text{,}\) if \(d=0\text{,}\) then \(c=0\text{,}\) contradicting the hypothesis that we are not in a degenerate case. The slope of \(\ell_2\) is then \(-c/d\text{.}\) Since \(ad-bc=0\text{,}\) the two slopes are then equal, and so the solution sets are either parallel and distinct (no solutions), or are the same (infinitely many solutions).
    ðŸ”—
    ðŸ”—
    Once you learn about Jordan--Gauss elimination, you will be able to give a purely algebraic proof of the statement above. In fact, this will be one of the exercises in one of your problem sheets. But I strongly encourage you to study the geometric proof above. A good geometric intuition is an invaluable tool for a mathematician.
    ðŸ”—

    Remark 1.1.15.

    A few remarks about this theorem are in order.
    1. This theorem says that the solution set of a linear systems like the one above is either empty (i.e. no solutions), infinite or consists of a single point. There is no linear system with two equations and two variables that has exactly two solutions. If you can find two solutions, you can find infinitely many. Actually, this property is true of every> linear system. (See Theorem TheoremÂ 1.1.16)
      ðŸ”—
      ðŸ”—
    2. Another curious aspect of this theorem is that it tells us that we can decide whether the system has exactly one solution or not simply by looking at the coefficients on the left-hand side of the equal signs. This means that \(b_1\) and \(b_2\) are irrelevant to the question of whether the system has exactly one solution. However, they are important when determining whether the system has none or infinitely many solutions. Again, this is a special case of a phenomenon that holds true for more general linear systems.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    For the remainder of the section, we will see a few more instances of the following theorem in systems with three equations. We will see a proof of this result much later in the course.
    ðŸ”—

    Theorem 1.1.16.

    The solution set of a linear system is either empty, infinite or consists of a single point.
    ðŸ”—
    ðŸ”—

    Example 1.1.17.

    Consider the linear system given by
    \begin{equation*} \left\{ \begin{alignedat}{7} 3x \amp\amp\; + \amp\amp\; 2y \amp\amp\; - \amp\amp\; z \amp = \amp\amp\; 0 \\ \amp\amp \amp\amp 2y\amp\amp\; + \amp\amp\; z \amp = \amp\amp\; 3 \\ \amp\amp \amp\amp \amp\amp \amp\amp 3z \amp = \amp\amp\; 6 \end{alignedat} \right. \end{equation*}
    Dividing both sides of the last equation by \(3\text{,}\) we immediately get \(z=2\text{.}\) We can now subtract the equation \(z=2\) from the second equation to get
    \begin{equation*} \left\{ \begin{alignedat}{7} 3x \amp\amp\; + \amp\amp\; 2y \amp\amp\; - \amp\amp\; z \amp = \amp\amp\; 0 \\ \amp\amp \amp\amp \amp\amp \amp\amp 2y \amp = \amp\amp\; 1 \\ \amp\amp \amp\amp \amp\amp \amp\amp z \amp = \amp\amp\; 2 \end{alignedat} \right. \end{equation*}
    Dividing both sides of the second equation by \(2\text{,}\) we get \(y=1/2\text{.}\) Finally, we can subtract each side of \(2y=1\) from each side of the first equation, and also add each side of \(z=2\) to each side of the same first equation. We obtain
    \begin{equation*} \left\{ \begin{aligned} 3x \amp = 1 \\ y \amp = \frac{1}{2} \\ z \amp = 2 \end{aligned} \right. \end{equation*}
    So, the only solution to this linear system is \((x,y,z)=(1/3,1/2,2)\text{.}\)
    ðŸ”—
    Geometrically, this is the intersection of three planes. The first plane is given by the equation \(3x+2y-z=0\text{.}\) We will see later in the course that this is the only plane that is perpendicular to the vector \((3,2,-1)\) and that contains the origin. The second plane is defined by the equation \(2y+z=3\text{,}\) and the third plane by \(3z=6\text{.}\) The only point of intersection of these three planes is the point \((1/3,1/2,2)\) that we found above.
    ðŸ”—
    ðŸ”—

    Example 1.1.18.

    Letâ€™s now find all the solutions to the linear system
    \begin{equation*} \left\{ \begin{alignedat}{7} 2x \amp\amp\; - \amp\amp\; y \amp\amp\; + \amp\amp\; 3z \amp = \amp\amp\; 1 \\ 4x \amp\amp\; - \amp\amp\; 2y \amp\amp\; + \amp\amp\; 6z \amp = \amp\amp\; 0 \\ \amp\amp\; \amp\amp\; y \amp\amp\; - \amp\amp\; 2z \amp = \amp\amp\; 1 \end{alignedat} \right. \end{equation*}
    Note that the expression \(4x-2y+6z\) is twice the expression \(2x-y+3z\text{.}\) Therefore, if we multiply both sides of the first equation by \(2\) and subtract each side from each side of the first equation, we end up with the second equation being \(0=-2\text{,}\) which is clearly absurd. This means that this linear system has no solutions.
    ðŸ”—
    Geometrically, the solution set of \(2x-y+3z=1\) describes a plane that is parallel and distinct to the plane formed by the solution set of \(4x-2y+6z=0\text{.}\) Therefore, they never intersect, so there is no common solution.
    ðŸ”—
    ðŸ”—

    Example 1.1.19.

    The next linear system is an example of a linear system of three linear equations in three variables that has no solutions but where, unlike the previous example, the system formed by any two equations has infinitely many solutions:
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp - \ampp z \amp = \ampp 1 \\ x \ampp + \ampp y \amp = \ampp 0 \\ y \ampp + \ampp z \amp = \ampp 0 \end{alignedat} \right. \end{equation*}
    Our first move will be to subtract the last equation from the second one. We obtain the equivalent system
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp - \ampp z \amp = \ampp 1 \\ x \ampp - \ampp z \amp = \ampp 0 \\ y \ampp + \ampp z \amp = \ampp 0 \end{alignedat} \right. \end{equation*}
    It is immediately clear from this new system that there are no solutions, since the first two equations are contradictory. However, looking back at our original system, it is easy to see that any two equations have infinitely many solutions. Take for example the first two:
    \begin{equation} \left\{ \begin{alignedat}{5} x \ampp - \ampp z \amp = \ampp 1 \\ x \ampp + \ampp y \amp = \ampp 0 \\ \end{alignedat} \right.\tag{1.1.5} \end{equation}
    This is a linear system with two equations and three unknowns. Note that if we allow \(x\) be any real number \(t\text{,}\) then we always get a solution. Indeed, we only need to set \(y=-t\) and \(z=t-1\text{.}\) In other words, the triple \((t,-t,t-1)\) is always a solution to the linear system (1.1.5), independently of the real number \(t\) that we choose.
    ðŸ”—
    ðŸ”—

    Example 1.1.20.

    Our final example in this section will be that of a system linear equations whose solution set is a two-parameter family. This will be a system of two linear equations on the variables \(x,y,z,w\text{.}\)
    \begin{equation} \left\{ \begin{alignedat}{9} x \ampp + \ampp y \ampp + \ampp z \ampp + \ampp w \amp = \ampp 0 \\ \ampp \ampp \ampp \ampp x \ampp + \ampp w \amp = \ampp 1 \end{alignedat} \right.\tag{1.1.6} \end{equation}
    Subtracting each side of the second equation from each side of the first, we obtain the equivalent system
    \begin{equation*} \left\{ \begin{aligned} y+z \amp = -1\\ x+w \amp = 0 \end{aligned} \right. \end{equation*}
    We can freely set \(x\) to be any real value \(t\text{,}\) and freely set \(y\) to be any real value \(s\) This leads to \(w=-t\) and \(z=-1-s\text{.}\) Therefore, the solution set of this system is given by the two-parameter solution
    \begin{equation*} x=t,\quad y=s,\quad z=-1-s,\quad w=-t\qquad (s,t\in\R)\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—

    Remark 1.1.21.

    Let me observe that the parametric solution \((t,s,-1-s,-t)\) can be decomposed as
    \begin{equation*} (t,s,-1-s,-t)=(0,0,-1,0) + t\cdot (1,0,0,-1) + s\cdot (0,1,-1,0)\text{.} \end{equation*}
    The geometric interpretation of this is the following. The solution set of the system (1.1.6) is the plane in \(\R^4\) that contains the point \((0,0,-1,0)\) and is generated by the two vectors \((1,0,0,-1)\) and \((0,1,-1,0)\text{.}\)
