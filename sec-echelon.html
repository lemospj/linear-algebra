    Section 1.2 The reduced row echelon form of a linear system

    I already mentioned in the previous section that we will make a systematic use of three operations to reduce a linear system to a simpler equivalent one. These are called elementary row operations. Let me remind you what they are:
    1. Multiplication of an equation in the system by a non-zero constant.
      ðŸ”—
      ðŸ”—
    2. Equation swapping.
      ðŸ”—
      ðŸ”—
    3. Addition of a constant multiple of an equation to another equation.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    When solving a linear system of equations, we do not need to carry around all the variables involved. We can instead organize all the coefficients in an array (which we will call a matrix) where each column corresponds to a variable, except for the last one, which will contain the numbers that appear on the right-hand side of the linear equations that form the system.
    ðŸ”—

    Definition 1.2.1. Matrix.

    If \(m\) and \(n\) are positive integers, an \(m\times n\) matrix is an array of number with \(m\) rows and \(n\) columns. If \(m=n\text{,}\) then we say that the matrix is a square matrix.
    ðŸ”—
    ðŸ”—
    An \(m\times n\) matrix is usually written in the form
    \begin{equation*} \begin{pmatrix} a_{11} \amp a_{12} \amp \cdots \amp a_{1n}\\ a_{21} \amp a_{22} \amp \cdots \amp a_{2n}\\ \vdots \amp \vdots \amp \ddots \amp \vdots \\ a_{m1} \amp a_{m2} \amp \cdots \amp a_{mn} \end{pmatrix} \end{equation*}
    where each \(a_{ij}\) (\(1\leq i \leq m\text{,}\) \(1\leq j\leq n\)) is a (real or complex) number. Note the similarity with the notation that was used to describe a generic linear system in the previous section: \(a_{ij}\) is the entry in row \(i\) and column \(j\text{.}\)
    ðŸ”—

    Example 1.2.2.

    For example,
    \begin{equation*} \begin{pmatrix} 1 \amp 3 \amp \pi \\ 0 \amp 1 \amp -3 \end{pmatrix} \end{equation*}
    is a \(2\times 3\) matrix whose entries are real numbers.
    ðŸ”—
    ðŸ”—

    Example 1.2.3.

    The matrix
    \begin{equation*} \begin{pmatrix} 1 \amp 0 \\ 0 \amp 1 \end{pmatrix} \end{equation*}
    is called the \(2\times 2\) identity matrix. In general, the \(m\times m\) identity matrix is the square matrix with \(1\)â€™s along the main diagonal and \(0\)â€™s everywhere else (i.e. \(a_{ij}=1\) if \(i=j\) and \(a_{ij}=0\) if \(i\neq j\)). For instance, the \(3\times 3\) identity matrix is the matrix
    \begin{equation*} \begin{pmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{pmatrix} \end{equation*}
    ðŸ”—
    ðŸ”—
    Now consider a system of linear equations in the variables \(x_1,\ldots,x_n\) with \(m\) equations:
    \begin{equation*} \left\{ \begin{alignedat}{9} \amp a_{11}x_1 \amp\amp+ \amp\amp a_{12}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{1n}x_n \amp\amp = \amp\amp \; b_1\\ \amp a_{21}x_1 \amp\amp + \amp\amp a_{22}x_2 \amp\amp+ \amp\amp\quad \ldots \amp\amp + \amp\amp \; a_{2n}x_n \amp\amp = \amp\amp \; b_2\\ \amp \quad\vdots \quad \amp\amp \quad \amp\amp\quad\vdots \quad \amp\amp \amp\amp\quad \ldots \quad \amp\amp \amp\amp\quad\vdots \quad \amp\amp \amp\amp\; \vdots\\ \amp a_{m1}x_1 \amp\amp + \amp\amp a_{m2}x_2 \amp\amp + \amp\amp\quad \ldots \amp\amp + \amp\amp\; a_{mn}x_n \amp\amp = \amp\amp \; b_m \end{alignedat} \right. \end{equation*}
    To this system we will associate the \(m\times (n+1)\) matrix \((m_{ij})_{ij}\) whose entry \(m_{ij}\) equals \(a_{ij}\) when \(j\leq n\) and equals \(b_i\) when \(j=n+1\text{:}\)
    \begin{equation*} \left( \begin{array}{cccc|c} a_{11} \amp a_{12} \amp \cdots \amp a_{1n} \amp b_1\\ a_{21} \amp a_{22} \amp \cdots \amp a_{2n} \amp b_2\\ \vdots \amp \vdots \amp \ddots \amp \vdots \amp \vdots\\ a_{m1} \amp a_{m2} \amp \cdots \amp a_{mn} \amp b_m \end{array} \right) \end{equation*}
    Note that each of the first \(n\) columns of this matrix corresponds to one variable. So, the first column corresponds to the variable \(x_1\text{,}\) the second column corresponds to the variable \(x_2\text{,}\) and so on. The last column is reserved for the \(b_i\)â€™s. Each of the \(m\) rows corresponds to an equation. I used a vertical line between the \(n\)th and \((n+1)\)th column as a visual aid to help us separate the columns associated to the elements of the linear system to the left of the equal signs from those associated to the elements to the right. This is called the augmented matrix associated to the linear system. The use of the term augmented is related to the fact that we assigned a column to the \(b_i\)â€™s.
    ðŸ”—

    Remark 1.2.4.

    Implicit in the definition of the augmented matrix associated to a linear system is the fact that an order for the variables has been chosen. Indeed, if we keep the same system but reorder the variables, we may end up with a different matrix. In most of what follows, the chosen order of variables will be obvious. In those cases where it is not, it will be explicitly described.
    ðŸ”—
    ðŸ”—

    Example 1.2.5.

    The augmented matrix associated to the linear system
    \begin{equation*} \left\{ \begin{aligned} 3x-2y \amp = -1\\ -y + z \amp = \pi \end{aligned} \right. \end{equation*}
    in the variables \(x,y,z\) is the matrix
    \begin{equation*} \left( \begin{array}{ccc|c} 3 \amp -2 \amp 0 \amp -1 \\ 0 \amp -1 \amp 1 \amp \pi \end{array} \right) \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 1.2.6.

    The augmented matrix associated to the linear system
    \begin{equation*} \left\{ \begin{aligned} x \amp =1\\ y \amp =0\\ z \amp =3 \\ w \amp =-4 \end{aligned} \right. \end{equation*}
    in the variables \(x,y,z,w\) (in this order) is the matrix
    \begin{equation*} \left( \begin{array}{cccc|c} 1 \amp 0 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 3 \\ 0 \amp 0 \amp 0 \amp 1 \amp -4 \end{array} \right) \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 1.2.7.

    If a set of ordered variables is fixed from the outset, we can conversely go from a matrix with at least two columns to the linear system of equations that has that matrix as its augmented matrix. For example, in the variables \(x,y,z\) the linear system corresponding to
    \begin{equation*} \left( \begin{array}{ccc|c} 0 \amp 2 \amp 1 \amp 1 \\ -1 \amp -2 \amp 0 \amp 0 \\ 2 \amp 2 \amp 2 \amp 2 \end{array} \right) \end{equation*}
    is
    \begin{equation*} \left\{ \begin{alignedat}{9} \ampp \ampp 2y \ampp + \ampp z \amp = \ampp 1 \\ \ampp -\ampp x \ampp - \ampp 2y \amp = \ampp 0 \\ 2x \ampp + \ampp 2y \ampp + \ampp 2z \amp = \ampp 2 \end{alignedat} \right. \end{equation*}
    ðŸ”—
    ðŸ”—
    What you should take away from this discussion is that the information contained in a linear system and in its associated augmented matrix is exactly the same.
    ðŸ”—

    Definition 1.2.8. Reduced row echelon form.

    The augmented matrix associated to a linear system is said to be in reduced row echelon form if it satisfies all the following properties:
    1. Every row consists entirely of zeroes or the first non-zero element is a \(1\text{.}\) This \(1\) is called the rowâ€™s leading \(1\).
      ðŸ”—
      ðŸ”—
    2. All rows consisting entirely of zeroes are grouped together at the bottom of the matrix.
      ðŸ”—
      ðŸ”—
    3. If two consecutive rows do not consist entirely of zeroes, the the leading \(1\) in the lower of the two occurs farther to the right than the leading \(1\) in the higher row.
      ðŸ”—
      ðŸ”—
    4. A column containing a leading \(1\) has zeroes everywhere else.
      ðŸ”—
      ðŸ”—
    The columns containing the leading \(1\)â€™s are called the pivot columns, and the positions where the \(1\)â€™s occur are called the pivot positions. A linear system is said to be in reduced row echelon form if its associated augmented matrix is in reduced row echelon form.
    ðŸ”—
    ðŸ”—

    Example 1.2.9.

    The matrix
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 2 \amp 0 \amp 2\\ 0 \amp 0 \amp 1 \amp 3\\ 0 \amp 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{array} \right) \end{equation*}
    is in reduced row echelon form. The pivot columns are \(1\) and \(3\) and the pivot positions are \((1,1)\) and \((2,3)\text{.}\) In the variables \(x,y,z\text{,}\) the corresponding linear system is
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp + \ampp 2y \amp = \ampp 2\\ \ampp \ampp z \amp= \ampp 3\\ \ampp \ampp 0 \amp = \ampp 0\\ \ampp \ampp 0 \amp = \ampp 0 \end{alignedat} \right. \end{equation*}
    The leading variables are \(x\) and \(z\text{.}\) The remaining variables are called free variables. Free variables can be treated as parameters, i.e. we may assign any real numbers to them and solve the equations in the system with respect to the leading ones. For instance, since \(y\) is a free variable, we may assign any real number \(t\) to it. We then necessarily have \(x=2-2t\) and \(z=3\text{.}\) Therefore, the solution set for this system is given parametrically by
    \begin{equation*} x=2-2t, \quad y=t,\quad z=3\qquad (t\in\R). \end{equation*}
    ðŸ”—
    ðŸ”—

    Definition 1.2.10. Leading and free variables.

    The variables in a consistent linear system in reduced row echelon form that correspond to pivot columns are called leading variables. The remaining variables are called free variables.
    ðŸ”—
    ðŸ”—
    Note that part of the definition of free and leading variables is the requirement that the linear system in question is consistent. If a consistent linear system has free variables, then it will have infinitely many solutions. Indeed, we will be free to assign any real value to these, and the values of the leading variables will then be uniquely determined.
    ðŸ”—
    Letâ€™s see another example of a linear system with infinitely many solutions.
    ðŸ”—

    Example 1.2.11.

    Consider the linear system
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp - \ampp z \amp = \ampp 2\\ \ampp \ampp y \amp = \ampp 1 \end{alignedat} \right.\text{.} \end{equation*}
    The associated augmented matrix is
    \begin{equation*} \left( \begin{array}{ccc|c} 1 \amp 0 \amp -1 \amp 2\\ 0 \amp 1 \amp 0 \amp 1 \end{array} \right)\text{.} \end{equation*}
    It is clear that this system is in reduced row echelon form. The pivot columns are columns \(1\) and \(2\) which correspond to variables \(x\) and \(y\text{,}\) respectively. So, by definition, \(x\) and \(y\) are the leading variables and \(z\) is the free variable. So we know that the system will have infinitely many solutions. The solution set will have to be described parametrically. To do that, we solve every equation with respect to the free variables. We end up with
    \begin{equation*} x=2+t,\quad y=1,\quad z=t \end{equation*}
    where \(t\in\R\text{.}\)
    ðŸ”—
    ðŸ”—

    Lemma 1.2.12.

    A consistent linear system in reduced row echelon form with \(n\) equations has at most \(n\) leading variables.
    ðŸ”—
    ðŸ”—

    Proof.

    Each equation corresponds to a row of the associated augmented matrix, and each row has at most one leading variable. Therefore, our system admits at most \(n\) leading variables.
    ðŸ”—
    ðŸ”—

    Example 1.2.13.

    Consider the matrix
    \begin{equation*} \left( \begin{array}{cc|c} 1 \amp 2 \amp 0\\ 0 \amp 0 \amp 1 \end{array} \right)\text{.} \end{equation*}
    This matrix is in reduced row echelon form. The corresponding linear system in the variables \(x,y\) is
    \begin{equation*} \left\{ \begin{alignedat}{5} x \ampp + \ampp 2y \amp = \ampp 0\\ \ampp \ampp 0 \amp = \ampp 1 \end{alignedat} \right. \end{equation*}
    which obviously has no solutions.
    ðŸ”—
    ðŸ”—

    Example 1.2.14.

    The matrix
    \begin{equation*} \left( \begin{array}{cc|c} 1 \amp 0 \amp 6\\ 0 \amp 1 \amp 0 \end{array} \right) \end{equation*}
    is also in reduced row echelon form. The corresponding linear system in the variables \(x,y\) is
    \begin{equation*} \left\{ \begin{aligned} x \amp = 6\\ y \amp = 0 \end{aligned} \right. \end{equation*}
    which has exactly one solution.
    ðŸ”—
    ðŸ”—
    As you can see from the examples above, linear systems in reduced row echelon form are easy to solve. (In a way, they are already solved!) In the next section, you will learn about a procedure known as Gauss--Jordan elimination to transform linear systems into equivalent linear systems in reduced row echelon form.
    ðŸ”—
    Before getting into it, letâ€™s familiarize ourselves a little bit more with the concept of reduced row echelon form. The most useful examples in this case are those of augmented matrices that are not in reduced row echelon form.
    ðŸ”—

    Example 1.2.15.

    The following matrices are not in reduced row echelon form:
    \begin{equation*} \left( \begin{array}{cc|c} 1 \amp 1 \amp 1\\ 0 \amp 1 \amp 0 \end{array}\right), \quad \left(\begin{array}{ccc|c} 0 \amp 1 \amp 3 \amp 4\\ 1 \amp 0 \amp 0 \amp 5 \end{array}\right), \quad \left( \begin{array}{ccc|c} 0 \amp 0 \amp 0 \amp 0\\ 1 \amp 3 \amp 0 \amp 2\\ 0 \amp 2 \amp 3 \amp 0 \end{array}\right), \quad \left(\begin{array}{cc|c} 1 \amp 3 \amp 1\\ 1 \amp 2 \amp 1 \end{array}\right)\text{.} \end{equation*}
    ðŸ”—
    Column \(2\) is a pivot column for the first matrix. However, that column contains non-trivial entries away from the pivot position.
    ðŸ”—
    The leading \(1\) in the first row of the second matrix appears in the second column, but the leading \(1\) in the second row appears in the first column, which clashes with condition \(3\) of the definition of reduced row echelon form. If the rows were swapped, the matrix would then be in reduced row echelon form.
    ðŸ”—
    The third matrix is not in reduced row echelon form for two reasons. First, we find a row consisting entirely of zeroes appearing above non-zero rows. Then the first entry in the last row is not a \(1\text{.}\)
    ðŸ”—
    Finally, the only pivot column in the last fourth matrix is column \(1\text{.}\) However, it contains non-zero elements in more than one position.
    ðŸ”—
