    Section 3.1 Vector Spaces

    Informally, a vector space is a set on which we can perform addition and multiplication by scalars.
    🔗

    Definition 3.1.1.

    A vector space is a non-empty set \(V\) equipped with a structure of vector addition (formally, this is a function \(V\times V\rightarrow V\)) and scalar multiplication (formally, this is a function \(\R\times V\rightarrow V\)) satisfying the following axioms:
    1. \(\bu+(\bv+\bw)=(\bu+\bv)+\bw\) for all vectors \(\bu,\bv,\bw\in V\text{.}\)
      🔗
      🔗
    2. \(\bu+\bv=\bv+\bu\) for all vectors \(\bu,\bv\in V\text{.}\)
      🔗
      🔗
    3. There exists an element \(\bz\in V\) such that \(\bv+\bz=\bv\) for all \(\bv\in V\text{.}\)
      🔗
      🔗
    4. For every \(\bv\in V\text{,}\) there exists an element \(-\bv\) such that \(\bv+(-\bv)=\bz\text{.}\)
      🔗
      🔗
    5. \(a(b\bv)=(ab)\bv\) for all vectors \(\bv\in V\) and scalars \(a,b\in \R\text{.}\)
      🔗
      🔗
    6. \(1\bv=\bv\) for all vectors \(\bv\in V\text{.}\)
      🔗
      🔗
    7. \(a(\bu+\bv)=a\bu+a\bv\) for all vectors \(\bu,\bv\in V\) and scalars \(a\in\R\text{.}\)
      🔗
      🔗
    8. \((a+b)\bv=a\bv+b\bv\) for all vectors \(\bv\in V\) and scalars \(a,b\in\R\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 3.1.2.

    Consider the set \(\R[x]\) of polynomials in \(x\) with coefficients in the real numbers. This is a vector space. A vector in this vector space is a polynomial with real coefficients. It is also infinite-dimensional. Indeed, if \(p_1,p_2,\ldots,p_n\) is any finite collection of polynomials in \(\R[x]\text{,}\) we may take the maximum from among the set \(\deg(p_1),\deg(p_2),\ldots,\deg(p_n)\text{.}\) Say that the maximum is \(m\text{.}\) Any linear combination of \(\lambda_1 p_1+\lambda_2 p_2+\ldots+\lambda_n p_n\) will have degree at most \(m\text{,}\) and so we will never be able to write \(x^{m+1}\) as a linear combination of \(p_1,\ldots,p_n\text{.}\)
    🔗
    Note that differentiation \(\frac{d}{dx}:\R[x]\rightarrow \R[x]\) is a linear map, and so is definite integration \(\int_a^b:\R[x]\rightarrow \R\text{.}\)
    🔗
    🔗

    Example 3.1.3.

    We may also restrict to the set of polynomials \(\R[x]_{\leq d}\) of degree less than \(d\text{.}\) This is a vector space of dimension \(d+1\text{.}\) A basis for this vector space consists of the polynomials \(1,x,x^2,\ldots, x^d\text{.}\)
    🔗
    We can use these basis to describe the derivative map. Let’s see this in the case of \(\R[x]_{\leq 3}\text{.}\) This is a \(4\)-dimensional vector space with basis \(1,x,x^2,x^3\text{.}\) To describe the matrix associated to \(\frac{d}{dx}\) in this basis, we only need to calculate the derivatives of each one of the elements belonging to the basis and write them as columns vectors in this basis. Since
    \begin{equation*} \frac{d}{dx}[1]=0,\quad\frac{d}{dx}x=1,\quad,\frac{d}{dx}x^2=2x\quad\text{and}\quad \frac{d}{dx}x^3=3x^2\text{,} \end{equation*}
    we conclude that the matrix of the derivative \(\frac{d}{dx}:\R[x]_{\leq 3}\rightarrow \R[x]_{\leq 3}\) with respect to this basis is
    \begin{equation*} \begin{pmatrix} 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 0 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0\end{pmatrix}\text{.} \end{equation*}
    A similar discussion holds for linear maps defined by definite integration.
    🔗
    🔗

    Example 3.1.4.

    Consider the set \(C^0(\R,\R)\) of continuous functions from \(\R\) to \(\R\text{.}\) This is a continuous vector space. A vector in this space is a continuous function. This vector space is infinite-dimensional. Indeed, it contains the infinite-dimensional subspace consisting of all polynomials with real coefficients.
    🔗
    We may also consider the set \(C^1(\R,\R)\) of differentiable functions from \(\R\) to \(\R\text{.}\) Differentiation provides a linear map \(\frac{d}{dx}:C^1(\R,\R)\rightarrow C^0(\R,\R)\text{.}\)
    🔗
    🔗

    Example 3.1.5.

    Let \(C^2(\R^n,\R)\) denote the set of twice-differentiable functions from \(\R\) to \(\R\text{.}\) This is again an infinite-dimensional vector space. One important map defined on this vector space is the Laplacian \(\Delta:C^2(\R^n,\R)\rightarrow C^0(\R^n,\R)\) given by
    \begin{equation*} \Delta(f)=\frac{\partial^2 f}{\partial x_1^2}+\frac{\partial^2 f}{\partial x_2^2}+\ldots+\frac{\partial^2 f}{\partial x_n^2}\text{.} \end{equation*}
    For example, when \(n=2\text{,}\) let’s consider the function \(f(x,y)=x^3y+xy^2+x-y\text{.}\) Then
    \begin{equation*} \Delta(f)=6xy+2x\text{.} \end{equation*}
    The function in the kernel of \(\Delta\) are called harmonic functions and play an important role in both physics and mathematics. A trivial family of examples of harmonic functions are polynomials of degree at most \(1\text{.}\) The function \(f(x,y)=e^x\sin(y)\) offers a less trivial example.
    🔗
    🔗

    Example 3.1.6.

    A homogeneous linear ordinary differential equation of order \(n\) is a differential equation of the form
    \begin{equation*} y^{(n)}+a_{n-1}(x)y^{(n-1)}+\ldots+ a_1(x)y'+a_0(x)y=0\text{,} \end{equation*}
    where the \(a_i(x)\) are continuous real functions from \(\R\) to \(\R\text{.}\) It can be easily checked that the solution set is a linear subspace of \(C^n(\R,\R)\text{,}\) the vector space of \(n\)-differentiable functions from \(\R\) to \(\R\text{.}\) What is harder to show, but is true nonetheless, is that the solution set is a vector space of dimension \(n\text{.}\) This implies that, in order to completely solve a differential system like the one above, all we need to do is to find \(n\) linearly independent solutions. All other solutions will then be linear combinations of these.
    🔗
    For example, consider the homogeneous linear ODE of order \(2\) given by
    \begin{equation*} y''+y'+y=0\text{.} \end{equation*}
    Note that the functions \(y_1(x)=e^{-x/2}\cos\left(\frac{\sqrt{3}}{2}x\right)\) and \(y_2(x)=e^{-x/2}\sin\left(\frac{\sqrt{3}}{2}x\right)\) are solutions. It is also easy to see that \(y_1\) and \(y_2\) are linearly independent. It follows that every solution to the differential equation above is of the form \(ay_1+by_2\text{,}\) where \(a\) and \(b\) are real numbers. In other words, the solution set to the that differential equation is spanned by \(y_1\) and \(y_2\text{.}\)
    🔗
    🔗

    Exercises Exercises

    1.

    Show that the solution set of a homogeneous linear ordinary differential equation of order \(n\) is a linear subspace of \(C^n(\R,\R)\text{,}\) the vector space of \(n\)-differentiable functions from \(\R\) to \(\R\text{.}\)
    🔗
    🔗

    2.

    Consider the linear subspace \(V\) of \(C^{\infty}(\R,\R)\) spanned by \(\sin(x)\) and \(\cos(x)\text{.}\) (\(C^{\infty}(\R,\R)\) is the vector space of infinitely differentiable functions from \(\R\) to \(\R\text{.}\))
    1. Show that \(\sin(x)\) and \(\cos(x)\) are linearly independent, and that therefore they form a basis \(B\) of \(V\text{.}\)
      🔗
      🔗
    2. Find the matrix of the derivative \(\frac{d}{dx}:V\rightarrow V\) in the basis \(B=(\sin(x),\cos(x))\text{.}\)
      🔗
      🔗

    1. Show that the subset \(W\) of convergent sequences is a linear subspace of \(V\text{.}\)
