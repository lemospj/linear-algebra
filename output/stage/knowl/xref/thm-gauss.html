Theorem 1.3.1. Gauss-Jordan elimination.

A linear system with \(m\) variables and \(n\) equations is equivalent to a linear system with \(m\) variables and \(n\) equations in reduced echelon form. Moreover, a homogeneous linear system with $m$ variables and $n$ equations is equivalent to a homogeneous linear system with $m$ variables and $n$ equations.
