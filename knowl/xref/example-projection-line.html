Example 2.5.4.

Consider the linear subspace of \(\R^2\) defined as the set of solutions of the equation \(x-2y=0\text{.}\) This is a line of slope \(1/2\) passing through the origin. Let’s also consider the point \((1,2)\text{,}\) which does not lie on this line. Our objective is to find the point in the line \(x-2y=0\) that lies closest to \((1,2)\text{.}\)
🔗
The distance between the point \((1,2)\) and the line \(x-2y=0\) is the length of the line segment that starts at the point \((1,2)\) and ends at the line \(x-2y=0\text{,}\) intersecting it at a right angle. This line segment is depicted in Figure 2.5.5 as a dashed black line.
🔗
Consider now the vector \(\bv=(1,2)\) and the vector \(\bu\) defined as the vector whose head lies in the point of intersection between the line \(x-2y=0\) and the black dashed line in Figure 2.5.5. This is called the projection of \(\bv\) onto the line \(x-2y=0\text{.}\) The head of this vector is precisely the point on the line that lies the closest to \((1,2)\text{.}\) Define \(\bu^{\perp}:=\bv-\bu\text{.}\) This last vector has the same length as the dashed black line of Figure 2.5.5 and is parallel to it.
🔗
In the figure above, \(\theta\) is the angle between \(\bu\) and \(\bv\text{.}\) Using a little trigonometry, we find that \(\lVert\bu\rVert=\lVert\bv\rVert\cos\theta\text{.}\) If \(\bw\) is any non-zero vector on the line \(x-2y=0\text{,}\) we have \(\bu=\lambda\bw\) for some \(\lambda\in\R\text{.}\) This means that
\begin{equation*} |\lambda|\lVert\bw\rVert=\lVert\lambda\bw\rVert=\lVert\bu\rVert=\lVert\bv\rVert\cos\theta\text{.} \end{equation*}
Therefore,
\begin{equation*} |\lambda|=\frac{\lVert\bv\rVert}{\lVert\bw\rVert}\cos\theta\text{.} \end{equation*}
If we denote the angle between \(\bv\) and \(\bw\) by \(\theta(\bv,\bw)\text{,}\) then \(\cos\theta=\cos\theta(\bv,\bw)\) precisely when \(\lambda\geq 0\text{.}\) Otherwise, \(\cos\theta=-\cos\theta(\bv,\bw)\text{.}\) It follows that
\begin{equation*} \lambda=\frac{\lVert\bv\rVert}{\lVert\bw\rVert}\cos\theta(\bv,\bw)\text{.} \end{equation*}
So we can express \(\bu\) as a scalar multiple of \(\bw\text{:}\)
\begin{equation*} \bu=\frac{\lVert\bv\rVert}{\lVert\bw\rVert}\cos\theta(\bv,\bw)\bw\text{.} \end{equation*}
The last thing we need to do is to somehow remove any reference to the angle \(\theta\text{.}\) This can be done using the following equality:
\begin{equation*} \bv\cdot\bw=\lVert\bv\rVert\lVert\bw\rVert\cos\theta(\bv,\bw)\text{.} \end{equation*}
If \(\bv\) and \(\bw\) are orthogonal, then we already know that this holds. If not, then \(\cos\theta(\bv,\bw)\neq 0\text{.}\) Since
\begin{equation*} \lambda\bv\cdot\bw=\bv\cdot\bu=(\bu+\bu^{\perp})\cdot\bu=\lVert\bu\rVert^2=\lVert\bv\rVert^2\cos^2\theta\text{,} \end{equation*}
we conclude that
\begin{equation*} \bv\cdot\bw=\lVert\bv\rVert\lVert\bw\rVert\frac{\cos^2\theta}{\cos\theta(\bv,\bw)}\text{.} \end{equation*}
But since \(\cos\theta=\pm\cos\theta(\bv,\bw)\text{,}\) it follows that \(\cos^2\theta=\cos^2\theta(\bv,\bw)\text{,}\) and the expression above simplifies to
\begin{equation*} \bv\cdot\bw=\lVert\bv\rVert\lVert\bw\rVert\cos\theta(\bv,\bw)\text{.} \end{equation*}
Therefore,
\begin{equation*} \bu=\frac{\bv\cdot\bw}{\lVert\bw\rVert^2}\bw \end{equation*}
for any non-zero vector \(\bw\) on the line \(x-2y=0\text{.}\)
🔗
Let’s choose, for example, \(\bw=(2,1)\text{.}\) Then \(\bv\cdot\bw=4\text{,}\) \(\lVert\bw\rVert^2=5\text{,}\) which means that the projection of \(\bv\) onto the line \(x-2y=0\) is the vector
\begin{equation*} \bu=\frac{4}{5}\colvec{2\\1}=\colvec{\frac{8}{5}\\ \frac{4}{5}}\text{.} \end{equation*}
