Theorem 2.7.14.

Let \(f:\R^n\rightarrow \R^n\) be a linear map. Let \(\bv_1,\bv_2,\ldots,\bv_m\) be a set of eigenvectors of \(f\) associated to the eigenvalues \(\lambda_1,\lambda_2,\ldots,\lambda_m\text{,}\) respectively. If the eigenvalues are all distinct, then the set formed by the eigenvectors \(\bv_1,\bv_2,\ldots,\bv_m\) is linearly independent.
🔗
🔗

Proof.

If \(m=1\text{,}\) the result is obvious. Let’s assume, for the sake of contradiction, that the set \(S=\{\bv_1,\bv_2,\ldots, \bv_m\}\) are linearly dependent. Upon reordering the vectors if needed, we may assume that \(\bv_1,\bv_2,\ldots,\bv_r\) is a minimal linearly dependent subset of \(S\text{.}\) Obviously, \(r\geq 2\text{.}\) Therefore, there are non-zero scalars \(a_1,a_2,\ldots a_r\) such that
\begin{equation*} a_1\bv_1+a_2\bv_2+\ldots+a_r\bv_r=\bz\text{.} \end{equation*}
Note that all scalars \(a_1,a_2,\ldots,a_r\) must be non-zero, because otherwise the set \(\bv_1,\bv_2,\ldots,\bv_r\) would not be a minimal linearly dependent subset of \(S\text{.}\) We may now apply \(A\) to both sides of the equation to obtain
\begin{equation*} \lambda_1a_1\bv_1+\lambda_2a_2\bv_2+\ldots+\lambda_ra_r\bv_r=\bz\text{.} \end{equation*}
If \(\lambda_1=0\text{,}\) the we get
\begin{equation*} \lambda_2a_2\bv_2+\ldots+\lambda_ra_r\bv_r=\bz\text{,} \end{equation*}
where none of the coefficients \(\lambda_ia_i\) is zero. But then \(\bv_2,\ldots, \bv_r\) is a subset of linearly dependent vectors of \(S\) of size \(r-1 < r\text{,}\) which is a contradiction.
🔗
If, on the other hand, \(\lambda_1\neq 0\text{,}\) we then have
\begin{equation*} \lambda_1a_1\bv_1+\lambda_2a_2\bv_2+\ldots+\lambda_ra_r\bv_r-\lambda_1(a_1\bv_1+a_2\bv_2+\ldots+a_r\bv_r)=\bz\text{.} \end{equation*}
But this is equivalent to
\begin{equation*} (\lambda_2-\lambda_1)a_2\bv_2+\ldots+(\lambda_r-\lambda_1)a_r\bv_r=\bz\text{,} \end{equation*}
which is again a non-trivial linear relation among a set of vectors of size less than \(r\) because the eigenvectors \(\lambda_i\) are pairwise distinct. This is a contradiction too. We are led to conclude that the set of vectors \(\bv_1,\ldots,\bv_m\) is linearly independent.
