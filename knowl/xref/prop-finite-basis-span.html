Proposition 2.4.16.

Let \(S\) be a set of vectors in \(\R^n\text{.}\) There is a finite subset \(T\subseteq S\) that is linearly independent and that satisfies \(\vspan(T)=\vspan(S)\text{.}\)

Define \(V:=\vspan(S)\text{.}\) Start by choosing a maximal subset \(T\) of vectors in \(S\) that is linearly independent (see the exercises). This means that if we add any other vector of \(V\) to \(T\) this set becomes linearly dependent. Since \(T\) is linearly independent, it contains finitely many vectors. (In fact, at most \(n\text{.}\)) We only need to show that \(\vspan(T)=V\text{.}\)

Of course, \(\vspan(T)\subseteq V\text{.}\) If there were a vector \(\bv\in V\) not in \(\vspan(T)\text{,}\) then \(T\cup\{v\}\) would be linearly independent. But this cannot happen because \(T\) is a maximally linearly independent subset of \(V\text{.}\) This means that \(V\subseteq \vspan(T)\text{.}\) Therefore, \(T\) spans \(V\text{.}\)
