Proposition 2.5.9.

Let \(\bv\) be a vector in \(\R^n\text{,}\) and let \(\bw\) be a non-zero vector of \(\R^n\text{.}\) There is a unique vector \(\bp\) in the line generated by \(\bw\) such that \(\bv-\bp\) is orthogonal to \(\bw\text{.}\)
🔗
🔗

Proof.

There is at least one vector \(\bp\) in the line \(\ell\) generated by \(\bw\) such that \((\bv-\bp)\cdot\bw=0\text{.}\) Indeed, if we define
\begin{equation*} \bp=\frac{\bv\cdot\bw}{\lVert\bw\rVert^2}\bw\text{,} \end{equation*}
we easily see that
\begin{equation*} (\bv-\bp)\cdot\bw=\bv\cdot\bw-\bp\cdot\bw=\bv\cdot\bw-\frac{\bv\cdot\bw}{\lVert\bw\rVert^2}\bw\cdot\bw=\bv\cdot\bw-\bv\cdot\bw=0\text{.} \end{equation*}
Now we need to show that there can be no other such vector in the line \(\ell\) generated by \(\bw\text{.}\)
🔗
Suppose that \(\bq\) is another vector in the line generated by \(\bw\) such that \((\bv-\bq)\cdot\bw=0\text{.}\) We then must have
\begin{equation*} 0=(\bv-\bp)\cdot\bu-(\bv-\bq)\cdot\bu=(\bq-\bp)\cdot\bu \end{equation*}
for every \(\lambda\in\R\text{.}\) Since both \(\bp\) and \(\bq\) are scalar multiples of \(\bw\) (this is what it means to lie in the line generated by \(\bw\)), the difference \(\bq-\bp\) is a scalar multiple of \(\bw\text{,}\) and so lies in the line generated by \(\bw\text{.}\) Therefore, a particular instance of the equation above is \((\bq-\bp)\cdot(\bq-\bp)=0\text{.}\) Bu this means that \(\lVert\bq-\bp\rVert^2=0\text{,}\) which is only possible if \(\bq-\bp=\bz\text{.}\) This means that \(\bq=\bp\text{.}\)
