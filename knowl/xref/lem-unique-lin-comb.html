Lemma 2.2.13.

Let \(\bw\) be a vector in \(\R^n\text{,}\) and let \(\bv_1,\ldots,\bv_m\in\R^n\) be a set of \(m\) linearly independent vectors in \(\R^n\text{.}\) Then there is at most one linear combination \(\lambda_1\bv_1+\ldots+\lambda_m\bv_m\) of these \(m\) linearly independent vectors such that
\begin{equation*} \bw=\lambda_1\bv_1+\ldots+\lambda_m\bv_m\text{.} \end{equation*}

Proof.

Note that there may well be no linear combination of the linearly independent vectors that represents \(\bw\text{.}\) The claim is that if there is one, that is the only one. To prove this, suppose that we have two linear combinations of \(\bv_1,\ldots,\bv_m\) representing \(\bw\text{:}\)
\begin{equation*} \lambda_1\bv_1+\lambda_2\bv_2+\ldots+\lambda_m\bv_m=\bw \end{equation*}
and
\begin{equation*} \mu_1\bv_1+\mu_2\bv_2+\ldots+\mu_m\bv_m=\bw\text{,} \end{equation*}
where \(\lambda_1,\lambda_2,\ldots,\lambda_m,\mu_1,\mu_2,\ldots,\mu_m\in\R\text{.}\) If we subtract the second linear combination from the first we end up with
\begin{equation*} (\lambda_1-\mu_1)\bv_1+(\lambda_2-\mu_2)\bv_2+\ldots+(\lambda_m-\mu_m)\bv_m=\bw-\bw=\bz\text{.} \end{equation*}
Since the set \(\bv_1,\ldots,\bv_m\) is linearly independent, we must have \(\lambda_i-\mu_i=0\) for all \(i\text{.}\) Put differently, \(\lambda_i=\mu_i\) for all \(i\text{.}\) This means that the two linear combinations are actually the same.
