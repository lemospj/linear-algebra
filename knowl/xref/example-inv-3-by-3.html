Example 2.8.8.

Consider the matrix \(A=\begin{pmatrix} 1 \amp 2 \amp -1 \\ 1 \amp 3 \amp 2 \\ 0 \amp 1 \amp 0\end{pmatrix}\text{.}\) The determinant of \(A\) is \(-3\text{.}\) Therefore, \(A\) is invertible. Let’s find its inverse.
\begin{align*} \left( \begin{array}{ccc|ccc} 1 \amp 2 \amp -1 \amp 1 \amp 0 \amp 0 \\ 1 \amp 3 \amp 2 \amp 0 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \amp 1 \end{array} \right)\amp\xrightarrow{R_2\to R_2-R_1}\left( \begin{array}{ccc|ccc} 1 \amp 2 \amp -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 3 \amp -1 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \amp 0 \amp 1 \end{array} \right) \\ \amp\xrightarrow{R_3\to R_3-R_2}\left( \begin{array}{ccc|ccc} 1 \amp 2 \amp -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 3 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp -3 \amp 1 \amp -1 \amp 1 \end{array} \right) \\ \amp \xrightarrow{R_3\to -R_3/3}\left( \begin{array}{ccc|ccc} 1 \amp 2 \amp -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 3 \amp -1 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 1/3 \amp -1/3 \amp 1/3 \end{array} \right)\\ \amp\xrightarrow{R_2\to R_2-3R_3}\left( \begin{array}{ccc|ccc} 1 \amp 2 \amp -1 \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp -2 \amp 2 \amp -1\\ 0 \amp 0 \amp 1 \amp 1/3 \amp -1/3 \amp 1/3 \end{array} \right) \\ \amp\xrightarrow{R_1\to R_1+R_3}\left( \begin{array}{ccc|ccc} 1 \amp 2 \amp 0 \amp 4/3 \amp -1/3 \amp 1/3 \\ 0 \amp 1 \amp 0 \amp -2 \amp 2 \amp -1\\ 0 \amp 0 \amp 1 \amp 1/3 \amp -1/3 \amp 1/3 \end{array} \right) \\ \amp\xrightarrow{R_1\to R_1-2R_2}\left( \begin{array}{ccc|ccc} 1 \amp 0 \amp 0 \amp 16/3 \amp -13/3 \amp 7/3 \\ 0 \amp 1 \amp 0 \amp -2 \amp 2 \amp -1\\ 0 \amp 0 \amp 1 \amp 1/3 \amp -1/3 \amp 1/3 \end{array} \right) \end{align*}
So,
\begin{equation*} A^{-1}=\begin{pmatrix}16/3 \amp -13/3 \amp 7/3 \\ -2 \amp 2 \amp -1\\ 1/3 \amp -1/3 \amp 1/3\end{pmatrix}\text{.} \end{equation*}
