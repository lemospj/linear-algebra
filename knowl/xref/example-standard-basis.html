Example 2.4.19.

The set of standard unit vectors \(\be_1,\be_2,\ldots,\be_n\) in \(\R^n\) is a basis of \(\R^n\) called the standard basis of \(\R^n\text{.}\) In order to show this, we need to show two things: first, we need to show that the set of standard unit vectors is linearly independent; after this, we need to show that this set spans \(\R^n\text{.}\)
