Lemma 2.3.15.

Let \(S\) be a linearly independent set of vectors in \(\R^n\text{.}\) If \(\bv\in\R^n\) is a vector such that \(S\cup\{\bv\}\) is linearly dependent, then \(\bv\) can be written as a linear combination of vectors in \(S\text{.}\)

Proof.

Since \(S\cup\{v\}\) is linearly dependent, there is a collection of vectors \(\bu_1,\bu_2,\ldots,\bu_m\in S\) and scalars \(a_1,a_2,\ldots,a_m,c\in\R\) not all \(0\) such that
\begin{equation*} a_1\bu_1+a_2\bu_2+\ldots+a_m\bu_m+c\bv=\bz\text{.} \end{equation*}
Note that we must have \(c\neq 0\text{,}\) for otherwise we would have a non-trivial linear relation among vectors in \(S\text{,}\) meaning that \(S\) itself would be linearly dependent. But if \(c\neq 0\text{,}\) we have
\begin{equation*} \bv = -\frac{a_1}{c}\bu_1 -\frac{a_2}{c}\bu_2-\ldots-\frac{a_m}{c}\bu_m\text{,} \end{equation*}
as we wanted.
